## 📏 核心交互规则

| 类别 | 规则详情 |
|------|----------|
| **提问策略** | 每次**只问一个**单一问题，绝对禁止多问题组合轰炸。 |
| **答疑记录** | 精简用户问题，仅保留核心疑惑；去除冗余废话。 |

| **图文关联** | 整理核心内容（案例、概念）时，**必须关联**对应的 EPUB 图表链接。 |
| **术语统一** | 严格遵守下方的术语规范，禁止使用混淆性旧词。 |

## 📝 笔记生成标准 (Note Structure)

每一节的精简笔记应包含以下 8 个标准模块：

1.  **核心思想** (文字): 1-2 句话阐述本节主旨。
2.  **关键概念** (定义): 本节核心术语的定义与解释（前置以辅助理解）。
3.  **核心逻辑/框架** (表格/流程): 理论推导、思维模型或操作步骤。
4.  **案例讲解** (表格+图片): 整理原文中出现的所有案例，必须图文结合。
5.  **实战要点** (列表/表格): 交易信号、判断准则、常见误区。
6.  **核心总结** (总结表): 本节内容的浓缩回顾。
7.  **导航链接** (链接): 上一节、下一节、目录。
8.  **图片引用** (Markdown): 罗列本节引用的 EPUB 图片链接。
9.  **关联文件**: 必须创建同名的 `_学习讨论.md` 文件，用于存放 Q&A 和测试内容。

## 🗣️ 术语规范 (Terminology)

在生成内容时，必须将左侧旧词/概念替换为右侧标准用词：

| 原始概念/旧词 | ✅ 标准用词 | 说明 |
|---------------|-------------|------|
| 需求爆发 / 长阳 | **放量大涨** | 成交量扩大 + 大幅上涨 |
| 供应爆发 / 长阴 | **放量大跌** | 成交量扩大 + 大幅下跌 |
| 异常高量 / 恐慌 | **天量大跌** | 底部恐慌抛售 |
| 向下反弹 | **下冲反弹 / Spring** | 价格下测支撑后的反弹 |
| 上升 / 上升趋势 | **上涨** | 简洁统一 |
| 下降 / 下行 | **下跌** | 简洁统一 |
| 回落 | **回调** | 统一为回调 |
| 强势反弹 | **放量反弹** | 强调量能 |
| 无力反弹 | **缩量反弹** | 强调量能 |
| 垂直需求柱 (VDC) | **放量大涨 (VDC)** | 保留缩写 |
| 垂直供应柱 (VSC) | **放量大跌 (VSC)** | 保留缩写 |
