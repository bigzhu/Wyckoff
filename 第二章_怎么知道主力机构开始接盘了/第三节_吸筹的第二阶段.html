第三节 吸筹的第二阶段

核心思想

吸筹第二阶段 = 震荡区中的拉锯战

二次测试成功后，市场进入吸筹第二阶段。CM(狗庄)的核心任务是控制价格上涨以便大量收购股票。这个阶段是CM(狗庄)使用疲劳战术的关键时期，通过反复打压、向上提价、终极震仓等手段，迫使公众抛售筹码。

关键原则：必须在震荡区的右手边进场（即测试过程确认吸筹之后）

定义

吸筹是CM建仓并期待价格进入牛市的过程。CM的眼光在长远盈利，收购公众已失去兴趣的股票。

案例（一）：疲劳战术完整演示

📊 图2-6：第二阶段吸筹案例

蜡烛形态含义信号
1供应压倒价格离开上方震荡区（超卖）⏳ 观察
2-3AR(自动反弹)反弹无力❌ 不进场
4成功二测熊市结束，吸筹开始✅ 底部确认
5上影线+放量CM在顶部打压❌ 吸筹未完
6-7反弹被打压希望破灭，被套者继续抛售⏳ 等待
8-9缓慢下滑最折磨阶段，成交低迷❌ 继续观察
11-12快速反弹被否定再次确认CM控盘⏳ 吸筹继续

关键认知：

  • 突破阻力后又返回 = 吸筹未完成
  • 无论多少抄底消息 = 都不能相信
  • 供应枯竭（缺量） = 底部成熟信号

案例（二）：高位支撑的形成

📊 图2-7：支撑位逐步提高

市场背景：

阶段现象含义
X到Y明显熊市价格持续下跌
超卖下跌急速无反抗供应压倒性力度
反弹缺量反弹需求无力，供应枯竭信号

关键讲解：

  • 支撑位逐步提高 = CM的收购价格在上升 = 低价筹码稀少 = 吸筹即将完成
  • 缺量下跌 ≠ "有人反抗但失败" ≠ 努力≠结果
  • 缺量下跌 = 供应已极度枯竭 = CM可以随意控制价格
  • 缺量信号 = 底部已足够纯净，接下来必然反弹

原则解析：

2. 判断吸筹是否完成

必须看到：

  • 上涨波 > 下跌波（力度和成交量）
  • 供应枯竭的信号（缺量）
  • SOS或JOC的出现

3. 进场后必看

  • ✅ 价格和成交量递增 → 牛市很强，进场正确
  • ❌ 放量阴线 → 大量供应出现，立刻出场！

4. 疲劳战术的真实目的

  • 第一阶段：恐慌抛售已获得底部筹码
  • 第二阶段：需要获得被套者手中的筹码（心理折磨）
  • 第三阶段：清除最后的浮供（终极震仓）

💡 实战要点

常见错误

错误原因后果
看到支撑就进不知道吸筹哪个阶段被困在震荡区
突破阻力就追不知道是否吸筹完成在派发高位接盘
忽视疲劳战术被心理折磨迫使出场在底部被扫出
不等二次测试急于抄底陷入高风险位置

实战判断标准

观察点判断标准交易意义
支撑位变化支撑位逐步提高吸筹即将完成
涨跌波对比上涨波>下跌波需求持续扩大
成交量特征缺量不创新低供应已枯竭
测试出现小蜡烛+缺量安全进场点

📌 核心总结

三个关键认识

1. 第二阶段 = 吸筹最关键阶段

  • 从"供应>需求"向"需求>供应"转变
  • CM使用疲劳战术强制转变
  • 需要时间和心理考验

2. 三个进场位置的本质

  • 第一个：最安全但价格不是最低
  • 第二个：比较安全且价格相对较低
  • 第三个：最便宜但风险最大

3. 关键判断标准

  • 支撑位上升 = 吸筹进行中
  • 缺量反弹 = 供应已枯竭
  • 右手边进场 = 风险最低的确认
