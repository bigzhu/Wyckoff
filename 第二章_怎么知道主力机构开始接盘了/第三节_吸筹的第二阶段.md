# 第三节 吸筹的第二阶段

## 📌 核心观点

**第二阶段 = 吸筹活动进入拉锯阶段**

从二次测试成功开始，市场进入**震荡区交易**。这个阶段CM的核心任务是：**控制价格上涨以便大量收购股票**。

关键原则：**必须在震荡区的右手边进场**（即测试过程确认吸筹之后）

---

## 📊 什么是吸筹？

### 定义

吸筹是CM**建仓并期待价格进入牛市**的过程。

### 特点

- **时间长**：吸筹可能持续数周至数月
- **目标明确**：CM眼光在长远盈利，不在蝇头小利
- **筹码来源**：公众已失去兴趣的股票，恐慌抛售而来
- **目的**：建立足够头寸，为将来的上升奠定基础

### 为什么需要震荡区？

```
二次测试成功 → 熊市终止
    ↓
但需要吸收足够筹码后才能上涨
    ↓
所以进入震荡区（拉锯阶段）
    ↓
这个区间也为将要形成的牛市建立平台
```

## 📈 吸筹过程中的供求关系变化

### 阶段别的供求变化

| 阶段         | 供应     | 需求     | 状态                  |
| ------------ | -------- | -------- | --------------------- |
| **第一阶段** | 大于     | 小于     | 恐慌抛售，CM接盘      |
| **第二阶段** | 基本平衡 | 基本平衡 | CM缓慢吸收            |
| **第三阶段** | 持续枯竭 | 开始超过 | SOS/JOC出现，牛市在即 |

### 第二阶段的特征

```
上涨波 > 下跌波（力度和成交量都要大于）
    ↓
说明：需求在持续扩大，供应在持续枯竭
    ↓
但仍未完全反转，还需要第三阶段验证
```

## 🚪 吸筹中的三个进场位置

### 第一个进场位置：SOS/JOC后的回测（最安全）

```
SOS或JOC出现
    ↓
价格反弹确认
    ↓
出现缺量回调
    ↓ 最安全的进场点（2和3位置）
唯一要求：回测必须小蜡烛+低成交量
```

**特点**：

- 风险最低
- 但此时价格已经不是最低点
- 但这正是最安全的代价

### 第二个进场位置：二次测试后的弹簧效应

**条件**：

1. 看到成功的二次测试（缺量）
2. 观察到上涨阳线力度和成交量 > 下跌阴线
3. 在震荡区底部出现弹簧效应（短暂下跌然后反弹）

**操作**：

- 如果弹簧效应**稍微突破支撑** → 可以直接进场
- 如果弹簧效应**大幅突破** + 放量 → 等待二次测试后再进

**特点**：

- 比SOS更早进场
- 风险相对较高（因为还没有SOS确认）
- 需要谨慎观察后续发展

### 第三个进场位置：终极震仓中进场

**特点**：

- 可以直接在震仓过程中进场
- 但风险相对更高
- 可以等待震仓后的二次测试再进（风险最低）

**原则**：
终极震仓和弹簧效应的进场方式相同（都需要观察后续二次测试）

### 进场后必看

```
进场后出现：
- ✅ 价格和成交量递增 → 牛市很强，进场正确
- ❌ 放量阴线 → 大量供应，立刻出场！
```

## 💪 CM为什么使用疲劳战术？

### 目的

在震荡区中，CM竭尽全力建仓。正常交易无法满足时，就使用**疲劳战术**：

1. **在顶部布置大量卖单** → 控制价格上涨 → 低价收购
2. **通过疲劳战术** → 迫使公众抛掉股票

### 三种方法（逐步升级）

#### 方法1：反复打压（最温和）

```
公众经历恐慌抛售，已经割肉很多
    ↓
进入漫长枯燥的震荡区
    ↓
每次反弹都被打压
    ↓
公众被折磨得绝望，逐渐投降
```

**心理过程**：希望 → 失望 → 绝望 → 割肉

#### 方法2：向上提价吸引抄底者（中等）

```
CM无法在低位收购足够股票
    ↓
把收购价格往上提
    ↓
吸引抄底者进场
    ↓
在顶部立刻阻止上涨，打压价格
    ↓
抄底者也被套，被迫割肉
```

**效果**：同时清除底部和顶部的卖盘

#### 方法3：终极震仓（最强烈）

```
前两种方法后，还有最坚持的公众
    ↓
CM使用终极震仓
    ↓
价格迅速穿透所有支撑
    ↓
摆出"继续大幅下跌"的假象
    ↓
最坚持的公众因心理崩溃而割肉
```

**这是对公众的最后一击**

### 三种方法的共同目的

```
清除所有公众的抛盘
    ↓
把所有流通股票吸收到CM手中
    ↓
供应彻底枯竭
    ↓
可以安心允许价格上升进入牛市
```

## 📝 第二阶段案例详解

### 案例（一）：疲劳战术的完整演示

**背景**：

- 蜡烛1：供应压倒性力度，价格离开上方震荡区（超卖）
- 蜡烛2-3：自动反弹（不能进场）
- 蜡烛4：成功的二次测试（熊市结束）

**吸筹震荡区过程**：

```
蜡烛5：CM在顶部打压
    ↓ 反弹失败，被套者希望破灭

蜡烛6-7：再次反弹被打压
    ↓ 公众开始绝望，连续抛售

蜡烛8-9：突破阻力（吸筹初期，供应还很多）
    ↓ 被打压回落（说明吸筹未完，不能进场）

蜡烛9-10：最折磨的阶段
    ↓ 缓慢枯燥下滑，成交量低迷
    ↓ 公众在绝望中继续抛售

蜡烛11-12：快速反弹被否定
    ↓ 再次确认CM控盘，吸筹继续
```

**关键认知**：

- 突破阻力后又返回 = 吸筹未完成
- 无论多少抄底消息 = 都不能相信，必须等待
- 供应枯竭（缺量） = 牛市的孕育才算成熟

### 案例（二）：高位支撑的形成

**标志**：
从底部反弹时，支撑位逐步提高，说明CM的收购价格也在逐步提高

```
最初的支撑很低（超卖底部）
    ↓
经过反弹和回调
    ↓
下一次回调时支撑位变高
    ↓ = CM已买不到那么便宜的货
    ↓ = 底部股票稀少
    ↓ = 吸筹即将完成
```

**供应枯竭的表现**：

- 连续下跌中：价格收在狭窄范围
- 成交量变化：保持平稳甚至缩量
- 实际含义：卖单在减少，供应在枯竭

## 🎯 关键原则

### 进场的首要原则

```
必须在震荡区的右手边进场
    ↓
即在测试过程确认吸筹之后
    ↓ 不是左手边（形成时期）
    ↓ 而是右手边（确认时期）
```

**为什么**？

- 左手边：吸筹方向不确定（可能失败）
- 右手边：吸筹已确认（风险最低）

### 判断吸筹是否完成

**必须看到**：

- 上涨波 > 下跌波（力度和成交量）
- 供应枯竭的信号（缺量）
- SOS或JOC的出现

**不能只看**：

- 价格低不低
- 反弹力度强不强
- 市场消息乐不乐观

## 📖 答疑区

_用户提出的问题和我的解答，以及对关键概念的澄清_

### Q1: JOC是什么？

**问题**：文中所说的JOC什么?

**解答**：

**JOC** = **Jump Over Creek**（跳过溪流），表示价格直接穿破之前的关键阻力水平（通常是震荡区上轨或之前的高点）。

**与SOS的区别**：
- **SOS**（强势信号）：向上测试，缺量反弹 → 温和确认
- **JOC**（跳过溪流）：直接穿破阻力 → 强势确认，需求爆发

两者都表示供应枯竭和牛市即将开始，但JOC比SOS更强势。

---

### Q2: 弹簧效应是什么？

**问题**：文中所说的弹簧效应是什么?

**解答**：

**弹簧效应** = 价格在底部支撑位短暂向下穿破，然后迅速反弹向上（像弹簧被压下又弹起）。

**含义**：
- 短暂下跌但反弹了 → 底部有强劲买盘
- 这正是CM在接盘的表现 → 是进场的好机会

**与二次测试的区别**：
- 二次测试：缺量小蜡烛回调 = 供应已枯竭的验证
- 弹簧效应：短暂穿破支撑后反弹 = CM接盘的强势表现

### Q3: 为什么蜡烛3虽然是阴线，但却是看涨信号？

**问题**：案例中蜡烛3虽然是明显的阴线，但却被说是看涨信号。这是为什么？

**解答**：

这个问题的关键在于理解**缺量下跌的真实含义**。

**错误理解**：
- 蜡烛3是"努力≠结果"的停止行为 ❌

**正确理解**：
```
蜡烛1已经释放了所有供应（天量）
    ↓
现在蜡烛3出现缺量阴线
    ↓ 说明：根本没有卖盘努力，价格就被打下来了
    ↓ = 供应已经极度枯竭
    ↓ = CM可以用很小的力量就控制价格
    ↓ = 底部已经足够纯净
```

**核心认知**：
- 缺量下跌 ≠ "有人在反抗但没成功"（这才是停止行为）
- 缺量下跌 = **供应已经稀少到CM可以随意控制的程度**
- 这正说明底部已经足够清洁，接下来必然反弹

所以蜡烛3是看涨信号，是因为**缺量本身就说明供应枯竭到极致**。

### 测试1：理解CM为什么必须使用疲劳战术

**我的问题**：

二次测试已经成功，供应在量上已枯竭。但为什么CM还要进入漫长的震荡区使用疲劳战术？二次测试后CM已经获得了什么信息？但还缺少什么，必须继续折磨？

**你的回答**：

初步：CM要在震荡中折磨散户，拿到低价筹码。

深化后：如果直接上升，被套的散户会产生希望，捏紧筹码不卖，CM无法在低价获得这些筹码。所以必须用疲劳战术让他们经历"希望→失望→绝望"，最后心理崩溃，低价卖出。

**我的评价和补充**：

✅ **理解正确的部分**：
- 完全抓住了疲劳战术的核心逻辑
- 理解了二次测试后还有"被套散户"这个关键群体
- 深刻认识到吸筹不仅是"量"的枯竭，还是"心理"的枯竭

📌 **关键要点**：
CM使用疲劳战术的真正目的 = 把所有筹码都从公众手中榨出来。二次测试只说明供应在"量"上枯竭，但被套的人心理上还有希望，必须通过折磨让他们从"心理"上也彻底投降。

### 测试2：三个进场位置的风险与收益权衡

**我的问题**：

如果看到成功的二次测试，会在三个位置中选择哪一个进场？关键是如何在"安全性"和"进场时机"之间权衡？

**你的回答**：

初步：选择位置3（终极震仓），因为价格最好。

深化后：在终极震仓中小额进场，获得最好的价格但承担较大风险。然后在SOS/JOC之后的回测时加仓，虽然价格不是那么好了，但有明确的供应枯竭信号，风险大幅降低，确定性大幅提高。

**我的评价和补充**：

✅ **理解正确的部分**：
- 完成了从"追求最便宜"到"风险-收益权衡"的思维升级
- 提出了分批进场的实战策略：小额进场+大额加仓
- 理解了不同位置的本质区别

⚠️ **关键深化**：
这不是简单的"两次进场"，而是一个更深的实战智慧：
- 第一批（终极震仓）：冒小风险换取最好价格
- 第二批（SOS/JOC回测）：用加仓的形式来验证前期进场是否正确

📌 **关键要点**：
威科夫理论的最高智慧不是教你"最低价在哪"，而是教你"如何在风险可控的前提下获得最好的风险-收益比"。

### 测试3：综合应用——识别吸筹完成的真正标志

**我的问题**：

如何判断吸筹是已经完成，还是还在进行中？需要观察什么来确认"吸筹已完成"？

**你的回答**：

初步：看到低量大跌出现，说明供应已经枯竭，很少成交量即可砸盘。关注是否还有供应来判断吸筹是否完成，CM一定会榨干所有供应才当作吸筹完成。

深化后补充的三点：
1. 支撑位逐步提高：低价已经买不到筹码，底部供应已枯竭，CM必须提高价格才能买到筹码
2. 上涨波 > 下跌波：供应枯竭，CM用力吸筹导致上涨强势，下跌无力
3. SOS或JOC的出现：最终确认底部供应是否消耗殆尽

**我的评价和补充**：

✅ **理解正确的部分**：
- 准确抓住了吸筹完成的核心指标："供应是否枯竭"
- 理解了"低量大跌"的含义：CM可以用很小力量控制价格
- 从多个维度理解吸筹完成的标志

📌 **深层理解**：
吸筹完成的三个维度确认：
- **数量维度**：支撑位逐步提高 = 低价筹码越来越少
- **意愿维度**：上涨波 > 下跌波 = 没人有货可卖了
- **心理维度**：SOS/JOC出现 = 没人再想卖了

📌 **关键要点**：
从"有供应"→"供应枯竭"的全过程识别，体现了你对威科夫"因果关系原则"的深刻理解。吸筹就是"因"的过程，当"供应彻底枯竭"时，"果"（牛市）就必然产生。
