# 第一节 熊市终止的市场行为

## 📌 核心观点

熊市中，无论价格跌的多深，首先要把急于抄底的想法放弃。在公众中流行的说法是"跌的越深，越应该抄底"，但这种基于情绪的判断属于赌博。

**正确的判断方式是：观察市场行为，而不是基于价格深度判断**。

---

## 📊 熊市终止的4个关键市场行为

熊市的终止过程从理论上讲是：**需求扩大后供应逐渐耗尽**。

从价量行为上讲，要经历4个市场行为（如图2-1）：

```
初次支撑 → 恐慌抛售 → 自动反弹 → 二次测试
  ↓         ↓         ↓         ↓
需求出现  需求吸收   空头回补  验证底部
```

### 1️⃣ 初次支撑

#### 定义

在熊市中，CM提前介入阻止价格下跌并提供暂时的支撑。

#### 关键特征

- **需求扩大**是能够抑制价格下跌的关键因素
- 表现形式：**扩大成交量使下跌停止**
- 初次支撑之前的熊市：持续而且无反抗的下跌过程

#### 深层含义

- 初次支撑是来自CM的需求进场
- 虽然暂时，但有可能把市场带入吸筹阶段
- 这种行为的本质是需求扩大，而不是形态

#### 理论根据

供过于求的背景必须转变为供求平衡和供不应求，熊市才能终止。而达到这一转变的最终力量是**需求扩大**。

在熊市中，公众处于恐慌，因为他们在顶部已经满仓，没有能力再买入。真正有能力接盘的只有CM，他们进场带来的需求才能使下跌停止。

#### 著名案例

**巴菲特的初次支撑案例（2008年金融危机）：**

- 时间：2008年9月
- 行动：斥资50亿美元以123美元/股买入高盛股票
- 背景：高盛股票从220美元/股一路跌下来
- 后续：巴菲特进场后，市场一片哗然认为他犯了大错
- 恐慌抛售：高盛继续大跌到50美元/股以下
- 结果：第二年高盛涨到了193美元/股

**最近案例（2016年石油股）：**

- 时间：2016年1月
- 行动：买入飞利浦66股票（石油股）
- 背景：国际石油市场正处于熊市
- 结果：到2016年3月，已经有每股10美元的利润

### 2️⃣ 恐慌抛售

#### 定义

公众因恐惧造成的抛售行为。价格跌到公众心理无法承受的情况下，无奈抛掉亏钱的筹码。

#### 触发原因

- 市场的悲观情绪和消息影响
- CM通过媒体制造悲观情绪
- CM利用价格变化（长阴或缓慢下跌）制造继续下跌的假象

#### 市场行为

- **供应来源**：公众的抛售
- **需求来源**：CM的接盘
- **谁占优势**：CM主导的行为总是最有优势

#### 价量特征

- **超长阴线** + **超高成交量**
- 有时是一天，有时是持续几天
- 如果没看到这种价量行为，说明恐慌抛售还没出现

#### 关键认知

- 恐慌抛售产生真正持久的牛市
- 没有恐慌抛售产生的牛市不会持续太久
- 市场上的股票从恐慌抛售开始慢慢转移到CM手中

#### 两种易引起恐慌的现象

1. **恐慌抛售现象本身**
2. **震仓**：这两种都包含价格迅速大幅下跌

### 3️⃣ 自动反弹

#### 定义

继恐慌抛售之后的一个正常市场行为。

#### 特点

- 反弹经常是空头回补
- 上涨不会持续太久
- 不代表熊市结束或牛市开始

#### 关键警告

- 不能在这里抄底
- 必须耐心等待二次测试给出的信息

#### 风险提示

恐慌抛售之后，走势也可能出现**窄幅横盘行为**，这种行为会导致熊市继续。

### 4️⃣ 二次测试（★ 关键！）

#### 定义

自动反弹结束后，价格进入测试过程。是决定熊市是否终止的**关键一步**。

#### 观察重点

**供应是否出现**

#### 成功的二次测试标准

```
成功二次测试 = 小蜡烛 + 缩量
```

- 价格波动和成交量与恐慌抛售相比都**非常小**
- 说明市场上的供应在减弱
- 导致熊市的停止

#### 失败的二次测试

- 成交量在测试过程中继续扩大
- 说明市场上浮动供应还在
- 导致价格突破支撑，然后熊市恢复

#### 进一步的测试

如果二次测试中成交量依然大，但低于恐慌抛售的成交量，需要等待进一步的二次测试，直到发现供应已经耗尽。

#### 深层含义

**成功二次测试 = 熊市停止的信号**

1. **确认恐慌抛售**：恐慌抛售首次出现后，我们不知道后面是否还有同样情况出现
2. **支撑有效性确认**：前面恐慌抛售所形成的支撑是安全的
3. **未来震仓预警**：一旦在后面的吸筹过程中出现急速跌破这个支撑，说明吸筹已经接近结束（属于终极震仓）

---

## 🔍 四个行为全部出现 = 熊市终止

如图2-1所示，自动反弹后的走势是：

- **小碎步式下跌**
- **伴随成交量递减**

这种行为表明熊市终止，因为：

- 像恐慌抛售那样的卖盘力度已经不存在
- 市场接下来会进入**震荡区**
- CM会在震荡区暗自吸筹

## 💡 核心认知

### 表象 vs 本质

初次支撑在图上有时不容易辨认，有时和恐慌抛售发生在一起。

**关键原则**：

- 我们需要掌握的是**原则**，而不是**形态**
- **原则是本质，形态是表象**
- 观察市场时，首先要从理论根据的角度观察走势
- 而不是盲目地依赖图形和指标

### 抄底的风险

在熊市中，如果没有看到需求进入，任何以下都属于高风险指标：

- 技术系统产生的抄底信号
- 新闻消息
- 技术指标

**正确的抄底依据**：

1. 看到初次支撑（需求出现）
2. 看到恐慌抛售（供应释放）
3. 看到自动反弹（正常反应）
4. 看到**成功的二次测试**（供应枯竭）

### Q1: 巴菲特为什么在初次支撑进场，而不等二次测试？

**问题**：
巴菲特为什么两次买入都是在初次支撑？他为什么不在二次测试时候买入？那时候的价格更好。难道他没学过Wyckoff？他不懂Wyckoff。

**解答**：

这个问题非常深刻，最初的回答有问题，用户的批判才是对的。

**关键认知**：

- 巴菲特确实不懂（或不用）威科夫理论
- 他用的是**价值投资**逻辑，完全不同于威科夫的**市场行为分析**
- 初次支撑的进场不是"精妙策略"，而是他看价格便宜就买了
- 恰好进场在初次支撑点，只是**事后诸葛亮**的巧合

**价值投资 vs 威科夫的本质区别**：

| 维度     | 价值投资（巴菲特）   | 威科夫理论         |
| -------- | -------------------- | ------------------ |
| 关注点   | 公司基本面、内在价值 | 市场行为、供求关系 |
| 进场依据 | 价格<内在价值        | 价量信号确认       |
| 进场时机 | 看到便宜就买         | 等待安全信号       |
| 心理要求 | 有强大信念           | 耐心观察           |

**对散户的启示**：

- 不要学巴菲特在初次支撑进场（你没有他的资金量和心理素质）
- 应该用威科夫理论等待：二次测试、SOS、终极震仓后的二次测试
- 这些位置价格虽然不是最低，但**风险最低**

---

### Q2: 震仓怎么震的？目的是什么？怎么操作？

**问题**：
CM怎样进行震仓操作？具体的目的是什么？技术面怎么操作？

**解答**：

**震仓的三个阶段**：

1️⃣ **疲劳战术（为震仓做准备）**

- 在区间顶部布置大量卖单 → 控制反弹
- 每次反弹都打压 → 反复摧残散户心理
- 让价格缓慢下滑 → 造成绝望感
- 时间：可能持续数周甚至数月

2️⃣ **终极震仓（真正的清理）**

- 价格迅速穿透所有支撑（没有任何抵抗）
- 摆出"继续大幅下跌"的假象
- 最坚持的散户因心理崩溃而割肉
- CM在这时完成吸收

3️⃣ **验证（震仓的二次测试）**

- 震仓后出现反弹
- 如果反弹是缺量 → 供应已枯竭，吸筹完成
- 如果反弹是放量 → 还有供应，继续压价

**震仓的三个目的**：

| 目的             | 具体含义                         |
| ---------------- | -------------------------------- |
| **吸收剩余股票** | 逼出最后死扛的散户，全部吸收     |
| **测试市场供应** | 通过压价来测试是否还有卖单       |
| **造成假象**     | 让散户以为还要继续大跌，从而割肉 |

**CM的资金优势**：

- 有现金随时挂买单接盘
- 有筹码随时砸盘打压
- 时间充足，不急着反弹
- 散户则时间有限，心理承受力有限

**散户如何应对**：

❌ 错误做法：看到支撑被破就马上割肉

✅ 正确做法：

1. 承认无法预知是否震仓
2. 等待反弹后观察成交量
3. 缺量反弹 = 吸筹完成，可以确认底部
4. 在SOS或终极震仓后的缺量回调才进场

**关键认知**：

- 最坚持的散户割肉的时刻 = 供应最枯竭的时刻 = 吸筹完成的时刻
- 这也是为什么成功的二次测试后市场才能进入牛市
- 震仓看起来是最坏的时刻，实际上是底部最深的确认

### 测试1：二次测试的含义

**我的问题**：
成功的二次测试（小蜡烛+缺量）说明了什么？为什么说它是"熊市停止的信号"？

**你的回答**：
小量小价格变动说明市场卖单少了 → 散户手上没筹码了 → 不能继续下跌

**我的评价和补充**：

✅ **理解正确的部分**：

- 准确观察了现象：成交量缺量 = 卖单减少
- 深刻理解了原因：散户在恐慌抛售时已经把能卖的都卖了
- 推导出了结论：无法继续下跌 = 熊市停止

⚠️ **深化理解**：
成功二次测试的关键是**供应枯竭**，而不仅仅是"卖单少"。供应枯竭意味着：

- 想继续下跌的卖压已经消失
- 底部已经被确认（不会再创新低）
- 市场可以进入横盘或上升阶段

📌 **关键要点**：
二次测试不是看价格低不低，而是看**还有没有卖家**。有卖家就有下跌压力，没卖家就没压力。

---

### 测试2：初次支撑的作用

**我的问题**：
为什么初次支撑必须在恐慌抛售之前出现？如果没有初次支撑，直接出现恐慌抛售，会怎样？初次支撑对整个底部形成有什么作用？

**你的回答**：

1. 初次支撑是市场对价格下跌的自然反应，连巴菲特都会觉得划算买入。如果没有初次支撑，说明市场完全没有需求，价格会持续下跌直至归零，类似meme币。
2. 初次支撑是"试探"，CM用它测试市场底部，可能成功也可能失败。
3. 初次支撑"展示需求"——没有人知道后续走势，所以存在为恐慌抛售做准备。
4. 从市场角度，初次支撑测试市场是否有需求、是否有转牛的可能。

**我的评价和补充**：

✅ **理解正确的部分**：

- 认识到初次支撑是"自然反应"而非阴谋
- 用meme币例子说明无需求直接归零的现实
- 理解初次支撑是"试探"而非"保证"
- 看到初次支撑"展示需求"的核心作用

⚠️ **关键深化**：
初次支撑最核心的作用是**显示市场有转机的可能性**：

- 无初次支撑 = 市场完全死亡 = 继续下跌至归零
- 有初次支撑 = 市场开始有需求 = 转牛成为可能

但初次支撑本身**不能确认底部**，因为它只显示"有需求"，不显示"供应是否枯竭"。

📌 **关键要点**：
初次支撑 = "需求信号"，二次测试 = "供应验证信号"。两者缺一不可。

---

### 测试3：初次支撑参与者的命运

**我的问题**：
如果有人在蜡烛1（初次支撑）进场，然后一直持有到蜡烛4（二次测试），他的心路历程会是什么？

**你的回答**：

1. 蜡烛2（恐慌抛售）来临时：咬牙坚持
2. 蜡烛3（自动反弹）时：眼看就要回本了，继续坚持
3. 蜡烛4（二次测试）时：完了，趁着还值点钱赶紧卖了，不然归零了

**我的评价和补充**：

✅ **理解正确的部分**：

- 真实捕捉了散户的心理变化：从坚持→希望→绝望
- 理解了人性的弱点：看到缺量就恐惧，觉得没机会了
- 这正是初次支撑参与者成为"炮灰"的真实写照

⚠️ **深层含义**：
这正是威科夫理论要揭示的市场真相：

- 初次支撑吸引散户进场（不知道后续有多黑暗）
- 恐慌抛售+自动反弹折磨散户（给虚假希望）
- 二次测试缺量**让散户最后放弃**（在最不该卖的时刻割肉）
- CM在二次测试时完成吸筹（散户的筹码已经全部割肉）

📌 **关键要点**：
成功二次测试之所以重要，就是因为**它正好是初次支撑散户放弃的时刻**。此时供应彻底枯竭，吸筹可以开始进入下一阶段。
