# 第四节 吸筹的第三阶段

## 📌 核心观点

**第三阶段 = 吸筹测试和确认阶段**

从吸筹中期进入到最后阶段，CM的核心任务发生了转变：**从被动吸收转为主动测试**。

关键目标：**确认震荡区是吸筹而不是派发，从而确认吸筹已经完成**。

---

## 📊 CM在第三阶段的任务

### 核心任务

```
吸筹是否结束？ → CM需要测试
            ↓
怎么测试？ → 使用终极震仓
            ↓
测试的目的是什么？ → 评估市场上的股票供应
                ↓
为什么要这样做？ → 确认底部股票已全被吸收
                ↓
然后呢？ → 允许价格进入牛市
```

### 终极震仓的三个任务

#### 任务1：吸收剩余股票

```
CM让价格迅速冲破所有支撑
    ↓
摆出"继续大幅下跌"的姿态
    ↓
这样可以吸收市场上剩下的股票
    ↓ （抄底者和坚持者被迫卖出）
```

#### 任务2：继续迫使弱者卖出

```
通过价格迅速下跌
    ↓
激发最后的恐慌心理
    ↓
弱持有者因为无法承受而卖出
    ↓
CM全部吸收
```

#### 任务3：造成继续下跌的假象

```
表面上看：价格在暴跌
实际上：CM在评估还有多少供应

一旦穿破支撑后的反弹不强
    ↓
说明供应已经极度枯竭
    ↓
市场就可以进入牛市了
```

---

## 🚪 终极震仓的表现形式

### 定义

深度跌破吸筹区间底部支撑，是CM用来：

1. 吸收剩余股票
2. 迫使最后的卖盘
3. 测试供应是否已枯竭

### 特点

```
表现：快速、猛烈、深度穿破支撑
    ↓
成交量：通常伴随成交量扩大
    ↓
心理冲击：对公众造成最大程度的心理打击
    ↓
目的明确：是一种有意识的计划性行为，不是恐慌
```

### 关键认知

**这是吸筹结束的信号！**

- 不是熊市继续
- 不是派发开始
- 而是吸筹的"最后一击"

---

## 🔄 弹簧效应（Spring Effect）

### 定义

终极震仓的温和版本：价格只是**轻微穿破**底部支撑，然后快速反弹。

### 含义

```
弹簧效应出现
    ↓
= 底部支撑有强大的买盘
    ↓
= CM的吸筹已基本完成
    ↓
= 供应已经极度枯竭
```

### 与终极震仓的比较

| 特征         | 终极震仓     | 弹簧效应   |
| ------------ | ------------ | ---------- |
| **穿破幅度** | 深度穿破     | 轻微穿破   |
| **速度**     | 快速暴跌     | 短暂下跌   |
| **心理冲击** | 最强         | 中等       |
| **含义**     | 底部不太纯净 | 底部很纯净 |
| **后续**     | 强力反弹     | 迅速反弹   |

---

## 📈 终极震仓/弹簧效应后的二次测试

### 为什么要有二次测试？

```
终极震仓/弹簧效应出现了
    ↓
但我们还不确定：供应是否真的枯竭？
    ↓
所以需要二次测试
    ↓
在下跌和反弹中重新验证供应情况
```

### 二次测试的目的

**确认吸筹真的完成了**

成功的二次测试 =

- 缺量反弹（小蜡烛+低成交量）
- 供应已经彻底枯竭
- CM可以安心让价格上升

### 二次测试的结果

#### ✅ 成功的二次测试

```
价格在高位反弹
    ↓
出现小蜡烛+低成交量
    ↓
= 供应已枯竭
    ↓
= 吸筹结束，牛市开始
```

#### ❌ 失败的二次测试

```
价格反弹时
    ↓
出现放量阴线（大量供应）
    ↓
= 市场上仍有卖盘
    ↓
= 吸筹还未完成
    ↓
= 需要继续测试或折磨
```

---

## 💡 关键原理：CM为什么使用终极震仓？

### 需要确认的信息

问题1：底部的股票是否全被吸收？
问题2：市场上还有多少浮动供应？
问题3：这个震荡区是吸筹还是派发？

答案来自：终极震仓后的市场反应

### 如何从反应来判断？

```
穿破支撑后的反弹强度
    ↓
弱反弹 = 没有卖盘了，供应枯竭 → 吸筹完成
强反弹 = 有很多卖盘，供应充足 → 吸筹继续
```

### 为什么不能直接问？

因为CM无法直接知道市场上还有多少待售股票，所以他们必须：

- 通过穿破支撑来激发卖盘
- 观察市场的反应
- 根据反应判断是否吸筹完成

    ↓
不要恐慌逃离
    ↓
这恰好说明吸筹即将结束
    ↓

### 原则2：进场要在二次测试中

```
终极震仓/弹簧效应出现
    ↓
价格会反弹
    ↓
反弹过程中出现二次测试
    ↓
如果是缺量反弹
    ↓
这是最安全的进场点
```

### 原则3：二次测试不能放量

```
关键判断标准：
成交量必须缩小
蜡烛必须变小
    ↓
如果出现放量
    ↓
说明还有大量供应
    ↓
需要继续等待
```

### 测试1：理解终极震仓为什么是吸筹完成而非熊市继续

**我的问题**：

为什么急速、猛烈、深度穿破吸筹支撑的下跌，不是熊市继续，而是吸筹即将完成的信号？CM这次穿破支撑的目的是什么？

**你的回答**：

初步：这个操作就是终极震仓，CM在测试吸筹是否结束。

深化后：CM穿破支撑后，关键是看反弹时的成交量：
- 低成交量反弹 = 下方已经没有供应 = 吸筹完成
- 高成交量反弹 = 下方还有大量供应 = 吸筹未完成

**我的评价和补充**：

✅ **理解正确的部分**：
- 完全抓住了终极震仓的本质：这是一个"测试"行为
- 理解了判断吸筹完成的核心标准：反弹时的成交量（而非下跌的成交量）
- 这是威科夫理论中"通过市场反应来判断供应"的高级应用

📌 **关键深化**：
穿破支撑本身不决定任何事，**反弹时的成交量**才是CM获得答案的方式：
- 缺量反弹 = 供应已枯竭 = 吸筹完成
- 放量反弹 = 供应充足 = 吸筹继续

---

### 测试2：终极震仓与疲劳战术的区别

**我的问题**：

在第二阶段，CM反复打压公众来消磨耐心。那反复打压和终极震仓有什么本质区别？为什么CM不能在第二阶段早期就做"最后一击"，而要等到第三阶段？

**你的回答**：

反复打压是消磨耐心，终极震仓是最后一击让人绝望。

深化后：如果在第二阶段早期就做最后一击：
- CM需要用大量筹码去砸盘（违背吸筹目的）
- 价格被砸太低 → 散户反而加仓
- CM白白浪费筹码，吸筹还要继续

**我的评价和补充**：

✅ **理解正确的部分**：
- 完全理解了为什么"时机"对终极震仓至关重要
- 认识到终极震仓不是"简单的砸盘"，而是需要在供应已接近枯竭的时刻进行
- 深刻理解了CM的成本考虑：早期做终极震仓会导致巨大代价

📌 **关键认知**：
终极震仓的威力不在"力度大"，而在"时机对"。在供应已经枯竭的时刻，一个轻微的穿破就能清楚地评估还剩多少供应。

### 测试3：综合应用——识别吸筹完成和安全进场

**我的问题**：

观察到穿破吸筹支撑的大幅下跌，需要判断：
1. 这次下跌代表什么？
2. 应该怎么做？
3. 关键的确认信号是什么？
4. 反弹时的三种不同情况说明什么？

**你的回答**：

1. 成交量大说明可能是终极震仓，需要观望确认
2. 应该观望，看反弹的表现
3. 关键信号是反弹时的成交量
4.
   - A（缺量反弹）= 吸筹完成
   - B（放量反弹）= 吸筹未完成
   - C（再次放量下跌）= 下方供应巨大

**我的评价和补充**：

✅ **理解正确的部分**：
- 认识到穿破支撑的下跌本身不能判断趋势，必须看反弹
- 正确判断了反弹成交量与吸筹完成的关系
- 理解了需要多次确认供应是否真正枯竭

⚠️ **关键纠正**：
穿破支撑成交量大 ≠ 熊市继续。这个下跌很可能就是终极震仓，放大的成交量是正常的。关键是**反弹时的成交量**才能最终判断。

📌 **正确逻辑**：
- 穿破支撑成交量大 = 正在测试
- 反弹成交量小 = 测试结果：供应枯竭
- 反弹成交量大 = 测试结果：供应充足

### 测试4：深度理解——终极震仓与弹簧效应的选择

**我的问题**：

对比两个场景：
- 场景A：大幅穿破（10%跌幅），放量，快速
- 场景B：轻微穿破（2%跌幅），温和，缓慢

分析：吸筹状态、原因、反弹表现、进场时机

**你的回答**：

1. A吸筹未完成，B吸筹接近完成
2. 被迫，这是下方供应量决定的
3. A大幅反弹，B轻微小量反弹
4. A需要二次确认后再进，B风险更低，可在反弹回撤时进场

**我的评价和补充**：

✅ **理解正确的部分**：
- 完全理解了两种穿破方式代表的不同吸筹状态
- **第2点是最高级的认知**：认识到这是"被迫"而非"主动选择"
- 这体现了威科夫理论的最高智慧：市场行为是由供应量决定的

📌 **深层认知**：
```
CM不是主动选择终极震仓或弹簧效应
    ↓
而是市场的供应量"迫使"他这样做
    ↓
供应充足 → 必须深度穿破才能评估完全
    ↓
供应枯竭 → 只需轻微穿破就能评估清楚
```

📌 **进场策略的完整链条**：
- 场景A：大幅穿破 → 大量反弹 → 需二次测试缺量 → 才能确认吸筹完成
- 场景B：轻微穿破 → 小量反弹 → 这次反弹本身就是最好的进场点

📌 **关键结论**：
进场的安全性取决于"供应是否已经枯竭"，而不是"价格下跌了多少"。
