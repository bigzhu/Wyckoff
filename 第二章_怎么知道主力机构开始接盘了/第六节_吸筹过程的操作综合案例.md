# 第六节 吸筹过程的操作综合案例

## 核心思想

通过三个不同周期的吸筹案例，展示从恐慌抛售到牛市开始的完整过程。

**关键技能**：

- 识别吸筹完成的标志
- 识别派发开始的警告信号
- 找到多个合理的进场点

### 图2-19：短期吸筹四阶段流程

![图2-19：短期吸筹案例](../wyckoff_content/OEBPS/Image00041.jpg)

| 阶段         | 表现                                | 含义             | 信号      |
| ------------ | ----------------------------------- | ---------------- | --------- |
| **恐慌抛售** | 蜡烛1：成交量大幅增加，价格急速下跌 | CM接盘，需求扩大 | ⏳ 观察   |
| **自然反弹** | 空头回补，反弹无力                  | 不是真正的买单   | ❌ 不进场 |
| **二次测试** | 第一次：高量 / 第二次：缺量         | 供应从充足→枯竭  | ⏳ 准备   |
| **死角交易** | 蜡烛越来越短，既无新低也无新高      | 供应极度稀少     | ✅ 进场   |

### 进场信号确认

| 信号    | 表现                    | 进场时机   |
| ------- | ----------------------- | ---------- |
| **SOS** | 长阳线+高成交量突破死角 | 等回测缺量 |
| **LPS** | 价格回到支撑，缺量反弹  | 即可进场   |
| **JOC** | 三个更高+递增成交量     | 等回测缺量 |

**关键要点**：回调时只要不是放量回调，牛市就不会受威胁。

### 三阶段特征

**第一阶段：初期恐慌**

- 天量快速下跌（恐慌抛售）
- 空头回补（自动反弹）
- 浮动供应还存在（二次测试缺量不明显）

**第二阶段：供应充足的震荡**

### 图2-22：持续反弹失败

![图2-22：供应压力信号](../wyckoff_content/OEBPS/Image00044.jpg)

| 现象                              | 含义         | 判断        |
| --------------------------------- | ------------ | ----------- |
| 多次反弹都无法离开区间            | CM吸筹未完成 | ⏳ 继续等待 |
| 反弹高点逐步下降（台阶式）        | 供应压力很大 | ⏳ 不可追高 |
| 成交量递增但无应有上升结果（SOT） | 典型熊市特征 | ⏳ 保持耐心 |

**特别提醒**："努力没有结果" = 典型熊市特征，说明吸筹还未完成

## 🚀 派发警告信号：天量上涨

### 现象与含义

**牛市中的天量上涨是什么？**

成交量突然大幅增加，通常伴随价格涨幅放缓

**为什么危险？**

| 正常牛市特征      | 天量出现时              |
| ----------------- | ----------------------- |
| ✅ 上涨不需要天量 | ⚠️ 供应大量涌入         |
| ✅ 成交量稳定递增 | ⚠️ 人们不用抬价就能买入 |
| ✅ 价格持续上升   | ⚠️ 价格上涨暂停         |

### 天量上涨的含义

- **公众的"抢购高潮"出现**
- **CM可能开始派发（出货）**
- **供不应求的局面被打破**

### 应对策略

| 情况     | 应该做什么                 |
| -------- | -------------------------- |
| 有仓位   | 天量是预警，考虑减仓或观望 |
| 无仓位   | 等待派发结束再进           |
| 确认派发 | 立刻出场，不要侥幸         |

## 💡 核心认知

### 三个关键理解

**1. 长期吸筹的特征**

- 多次反弹都失败
- 反弹高点逐步下降（台阶式）
- SOT现象（努力无结果）明显
- 需要极其耐心等待

**2. 派发与吸筹的区别**

| 特征   | 吸筹     | 派发           |
| ------ | -------- | -------------- |
| 成交量 | 逐步增加 | 突然增加       |
| 价格   | 缓慢上升 | 快速上升后停滞 |
| 心理   | 折磨公众 | 吸引公众       |
| 操作   | 不着急   | 急着出货       |

**3. 多个进场点的意义**

- 不追求最完美的点
- 而是追求最安全的点
- 从蜡烛6→蜡烛12，风险逐步降低
- 确定性逐步增加

## 💡 实战要点

### 必须记住的四个规则

| 规则         | 含义                     | 后果               |
| ------------ | ------------------------ | ------------------ |
| 死角进场     | 蜡烛最短、既无新低无新高 | 最低风险           |
| 拒绝天量上涨 | 成交量突然爆增           | 派发警告，该出场   |
| 耐心等待二测 | 第一次测试失败不要放弃   | 第二次测试可能缺量 |
| SOT不可追高  | "努力≠结果"的现象        | 继续下跌可能性大   |

### 关键术语

| 术语         | 含义                                     |
| ------------ | ---------------------------------------- |
| **死角**     | 最低风险进场点（蜡烛越来越短）           |
| **SOT**      | Sign of Weakness（努力无结果，熊市特征） |
| **LPS**      | Last Point of Supply（最后供应点）       |
| **天量上涨** | 派发警告信号                             |
| **抢购高潮** | 公众疯狂买入，CM派发机会                 |

---

## 📌 核心总结

### 三个案例的共同规律

**短期吸筹**：快速→二测→死角→SOS/JOC

**长期吸筹**：缓慢→SOT→终极震仓→SOS/JOC

**进场逻辑**：一切都指向"供应是否枯竭"

### 最重要的警示

- 天量上涨 = 派发开始，该出场
- 反弹失败 = 供应充足，继续等待
- 蜡烛变短 = 供应枯竭，可以进场
