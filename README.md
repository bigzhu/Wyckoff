### 威科夫理论简介

威科夫理论（Wyckoff Method）是由查尔斯·威科夫（Charles Wyckoff）开创的技术分析体系。它不同于传统的技术指标分析，而是通过分析**价格、成交量和走势速度**，来理解市场背后的**供求关系**和**主力机构的意图**。

**核心思想**：
- 市场由主力机构（聪明钱）和公众组成
- 主力在底部吸筹，顶部派发
- 通过供求关系分析可以识别主力意图
- 在关键位置提前介入，获取最大利润

## 💡 核心要点速览

### 三大工具
```
价格 + 成交量 + 走势速度
   ↓
供求关系判断
   ↓
CM 意图识别
   ↓
交易决策
```

### 三大原则
1. **供求关系原则** - 需求>供应=上涨，供应>需求=下跌
2. **因果关系原则** - 趋势反转前需要准备过程（吸筹或派发）
3. **努力和结果原则** - 努力≠结果=停止行为=反转前兆

### 最重要的三个信号
```
✅ 吸筹完成 → 可以买入
❌ 派发开始 → 要出场
⚠️ 放量滞涨/滞跌 → 反转前兆
```
