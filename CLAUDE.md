## 📖 笔记整理方法（基于第一节优化）

### 目标
从冗长的学习记录（1600+行）→ 精简的查阅笔记（180行）

### 笔记结构（8 个模块）

| 模块 | 内容 | 形式 | 用途 |
|------|------|------|------|
| **核心思想** | 1-2 句话说清楚主旨 | 文字 | 快速理解节点要点 |
| **关键概念** | 核心术语定义与解释 | 定义 + 含义 | 辅助理解后续内容 |
| **核心逻辑/框架** | 理论推导/思维模型/步骤 | 表格/流程 | 掌握核心方法论 |
| **案例讲解** | 原文中出现的所有案例 | 表格 + 图片 | 理解理论如何应用 |
| **实战要点** | 信号/判断准则/误区 | 列表 + 表格 | 指导实盘操作 |
| **核心总结** | 整节内容的浓缩 | 总结表 | 复习回顾 |
| **导航链接** | 前后节点连接 | 链接 | 快速切换 |
| **图片引用** | EPUB 图表链接 | Markdown | 对照理论学习 |

### 精简原则

**保留**：
- ✅ 核心方法和框架
- ✅ 典型案例讲解
- ✅ 关键概念定义
- ✅ 实战应用要点
- ✅ EPUB 图表引用

**删除（单独存储）**：
- ❌ 详细的 Q&A 答疑区
- ❌ 详细的测试记录
- ❌ 过多的解释性文字
- ❌ 学习进度 checklist
- ❌ 重复的概念说明

### 表格化信息的优势

1. **快速扫读** - 关键信息一目了然
2. **便于对比** - 不同情况/案例的横向比较
3. **易于复习** - 表格形式方便快速回顾
4. **结构清晰** - 避免长篇论述的冗长感
5. **易于维护** - 更新和补充都比较方便

### 文件组织建议

```
每一节应创建：

📄 第X节_聪明钱的看盘顺序.md（精简笔记）
   ├─ 核心思想
```

## 📝 术语规范

### 统一的用词标准

| 概念         | 标准用词          | 说明                   |
| ------------ | ----------------- | ---------------------- |
| 需求爆发     | 放量大涨          | 成交量扩大 + 大幅上涨  |
| 供应爆发     | 放量大跌          | 成交量扩大 + 大幅下跌  |
| 异常高成交量 | 天量大跌          | 用于恐慌抛售等异常场景 |
| 向下反弹     | 下冲反弹 / Spring | 价格下测支撑后的反弹   |
| 价格形态     | 大涨 / 大跌       | 只描述价格涨跌幅度     |
| 成交量扩大   | 放量              | 相对于前期的成交量增加 |
| 成交量减少   | 缩量              | 相对于前期的成交量减少 |

### 同义词收束清单

| 旧用词              | 替换为        | 原因                         |
| ------------------- | ------------- | ---------------------------- |
| 长阳 / 放量长阳     | 放量大涨      | 统一术语，避免混淆           |
| 长阴 / 放量长阴     | 放量大跌      | 统一术语，避免混淆           |
| 垂直需求柱(VDC)     | 放量大涨(VDC) | 标准化表述，保留VDC缩写      |
| 垂直供应柱(VSC)     | 放量大跌(VSC) | 标准化表述，保留VSC缩写      |
| 强势反弹            | 放量反弹      | 更精确的量价描述             |
| 强势上涨            | 放量大涨      | 更精确的量价描述             |
| 无力反弹            | 缩量反弹      | 与放量反弹对应               |
| 高量上影线          | 放量大跌      | 简化表述                     |
| 大幅上涨            | 大涨          | 更简洁                       |
| 大幅下跌            | 大跌          | 更简洁                       |
| 下跌幅度            | 跌幅          | 简化表述                     |
| 上涨幅度            | 涨幅          | 简化表述                     |
| 上升 / 上升趋势     | 上涨          | 与下跌对应，更简洁           |
| 下降 / 下行         | 下跌          | 统一用词，与上涨对应         |
| 回落                | 回调          | 两者意思相近，统一为回调     |
| K线                 | 蜡烛          | 威科夫标准术语               |
| 买入                | 进场          | 威科夫标准术语               |
| 蜡烛幅度 / 波动幅度 | 蜡烛长度      | 更直接的表述                 |
| 底部                | 底            | 更简洁，如特指"底部形成"保留 |

**原则**：

- 避免多个同义词混用（如"长阳""长阴""垂直需求柱"混用）
- 用词应清晰指向具体含义
- 减少读者的思维转换成本

## 🔑 核心概念速查

### 三大工具

- **价格** - 蜡烛的长度、高低点位
- **成交量** - 成交量的高度、变化速度
- **走势速度** - 上涨或下跌的斜率

### 三大原则

- **供求关系** - 不平衡产生趋势（需求>供应=涨，反之=跌）
- **因果关系** - 因（准备过程）→ 果（趋势形成）
- **努力和结果** - 努力≠结果=停止行为=反转信号

### 关键术语

| 术语            | 含义                             |
| --------------- | -------------------------------- |
| CM              | 聪明钱/主力机构                  |
| 停止行为        | 努力没有产生相应结果的现象       |
| 垂直需求柱(VDC) | 长阳+大成交量（底部）            |
| 垂直供应柱(VSC) | 长阴+大成交量（顶部）            |
| 吸筹/派发       | CM的建仓/出货过程                |
| 死角            | 上升趋势的上轨与水平阻力线的交点 |
| 震仓            | CM为清除弱持有者而进行的价格下压 |
| 低量测底        | 卖单流入后供应枯竭的现象         |
