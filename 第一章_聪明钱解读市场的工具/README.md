# 第一章 聪明钱解读市场的工具是什么？

## 📚 章节概述

这一章是威科夫理论学习的**基础和核心**。它介绍了聪明钱（主力机构）如何观察市场、如何理解价格走势的基本方法论。

### 核心思想

> 聪明钱使用 **价格、成交量、走势速度** 这三个工具，通过分析 **供求关系** 来理解市场，而不是依赖技术指标。

## 🔑 核心概念速查表

### 三大工具
| 工具 | 说明 |
|------|------|
| 价格 | 蜡烛的长度、高低点位 |
| 成交量 | 成交量的高度、变化速度 |
| 走势速度 | 上涨或下跌的斜率 |

### 三大原则
| 原则 | 核心 | 应用 |
|------|------|------|
| 供求关系 | 不平衡产生趋势 | 判断强弱方向 |
| 因果关系 | 因→果 | 避免盲目操作 |
| 努力和结果 | 努力≠结果 | 识别反转信号 |

### 关键术语

| 术语 | 含义 |
|------|------|
| **CM** | 聪明钱，主力机构 |
| **超卖** | 价格持续急速下跌，无反弹 |
| **恐慌抛售(SC)** | 下跌中的潮水般抛售 |
| **停止行为** | 努力没有产生相应结果 |
| **放量大涨(VDC)** | 放量 + 大涨 |
| **放量大跌(VSC)** | 放量 + 大跌 |
| **抢购高潮(BC)** | 牛市顶的疯狂抢购 |
| **SOS** | Sign of Strength，需求强信号 |
| **SOW** | Sign of Weakness，供应弱信号 |
| **吸筹** | CM的建仓过程 |
| **派发** | CM的出货过程 |

## 💡 实战要点

### 看盘的 5 步法
```
1. 背景 → 确定大趋势背景
2. 价量形态 → 观察最近行为
3. 性质分析 → 理解行为含义
4. 结论预测 → 推断 CM 意图
5. 措施行动 → 交易决策
```

### 最重要的三个信号
```
✅ 吸筹完成：停止行为 + 需求扩大 = 可以进场
❌ 派发开始：供应扩大 + 需求减弱 = 要出场
⚠️ 危险信号：放量滞涨 / 放量滞跌 = 反转前兆
```

### 常见错误
```
❌ 只看技术指标 → 指标滞后，会被套
❌ 急着抄底 → 没有吸筹过程，结果被套
❌ 看到放量大涨就买 → 可能是 CM 在派发
❌ 把支撑当线 → 支撑是供求关系，不是线
```
