# 第三节 供求关系

## 📌 节点概述

本节深入讲解供求关系这个最核心的理论。供求关系不仅是解释市场运动的根本原理，也是区别威科夫理论和传统技术分析的关键差异。

---

## 核心要点

供应和需求是经济活动中最基本的原理。任何市场的价格波动都来自于供求关系的变化。我们常见的市场表象（支撑、阻力、趋势等）都是由供求关系的不平衡产生的。

**关键理解**：

- 需求 = 购买力
- 供应 = 流入市场的卖单数量
- 供应>需求 = 下跌趋势
- 需求>供应 = 上涨趋势
- 供应≈需求 = 震荡区（容易亏钱的危险区）

---

## 一、供求关系主导趋势的行为

### 简单例子

假设你要下单买 1000 股某只股票，价格是 100 元一股。但市场在这个价位只有 200 股。

```
你的需求：1000 股
市场的供应：200 股
→ 供不应求

经纪公司为了成交，分别以：
- 101 元成交 300 股
- 102 元成交 400 股
- 103 元成交 100 股

结果：价格上涨产生于供求关系的不平衡
```

### 市场含义

从供求关系理解市场的本质：

```
如果掌握了供求关系，即使十次交易只有五次正确，甚至三四次正确，
我们依然能盈利：
- 通过识别供求关系把风险降到最低
- 一次盈利的交易可以补平多次亏损
- 其他盈利交易就是纯盈利
```

**本书的目标**：通过逐根蜡烛分析，反映价格背后的供求关系。懂得了市场的供求关系，也就懂得了市场的本质。

---

## 二、证券市场的本质和表象

### 表象 vs 本质

**表象**：

- 走势图、技术指标（MACD、均线等）
- 几何图形、蜡烛形态
- 很多人依赖这些指标来交易
- **问题**：不断调整参数，市场又变了，陷入恶循环

**本质**：

- 供求关系（市场的第二层面信息）
- 趋势背后的动力
- 需求和供应的对比

### 正确的判断方法

例如：判断牛市结束需要什么条件？

```
理论基础：
1. 需求减弱
2. 供应大量进入市场
3. 需求没有足够力量继续主导市场

这些理论必须从价量行为中体现出来
我们要从直观的图上解读出这些理论
```

### 实战优势

如果掌握了如何从价量行为细节看出供求关系，我们就可以：

✅ 在市场趋势起来**之前**识别出即将出现的趋势
✅ 在危险来临**之前**做好出场准备
✅ 在市场有大的动作**之前**制订交易计划
✅ **实时掌握市场的脉搏**

**这是威科夫理论与传统技术分析的根本区别**

## 三、供求关系和上涨趋势

### 上涨趋势的本质

在上涨趋势中，价格上涨背后的原因是：**供不应求**

```
现象：
- 在同一价格，卖单无法满足增加的购买力
- 买方只能去买更高价位的股票
- 结果：价格不断抬高

这个过程中：
供不应求 → 需求>供应 → 价格上涨
```

### 牛市停止的信号

要使上涨停止或反转，**供求关系必须有变化**：

```
理论基础：
- 人们的购买力完全得到满足
- 不需要再抬高价格来满足需求
- 牛市开始停止

图表表现：
- 虽然参与进场者众多（高成交量）
- 但上涨没有取得多大进展
- 说明供应已经开始增加
  （来自CM的抛盘，因为他们的利润目标已达到）

市场背景的变化：
供不应求 → 供需平衡 → 供过于求
```

**关键意义**：这种市场本质的变化，**指标和软件无法提前告诉我们**！

### 为什么上涨中会出现回调？

```
原因：需求逐步减弱，市场必须降价来刺激新需求

牛市中回调的关键观察：
↓
这个回调是否引起供应扩大（CM抛盘现象）？

✅ 有供应扩大：
   - 回调对市场造成压力增大
   - 市场要恢复上涨趋势，必须有足够新需求
   - 新需求必须吸收回调带来的巨大卖压
   - 并以持续增加的数量恢复上涨趋势

❌ 如果回调后没有更大的新需求：
   - 主力抛盘会继续
   - 市场逐步进入派发阶段
   - 最终进入熊市
```

## 四、供求关系和下跌趋势

### 下跌趋势的本质

在下跌趋势中，**供应>需求，或需求很弱**

```
卖方视角：
- 找不到足够的客户
- 只能不断降价来刺激需求

买方视角：
- 如果价格没跌到满意的价格区
- 就不会进场
- 等待更低的价格
```

### 下跌趋势停止的条件

要让下跌趋势停止，**需求必须大量出现**：

```
必要条件：
1. 需求大量出现
2. 吸收掉当前市场的供应
3. 卖方找到了充足的买家
4. 不用再降价也能销售

结果：
- 价格跌势得到缓解
- 供应逐步减少
- 价格进入盘整

之后：
- CM在盘整区内开始建仓吸筹
- 或者准备再次派发
- 吸筹和派发都是下一波趋势的准备过程
```

### 安全抄底的理论根据

```
判断抄底的理论基础：
1. 供应枯竭
2. 需求增加并主导市场

能够反映出这个理论的价量关系：
1. 天量抛售（供应达到极限）
2. 低量测底（供应逐步减弱）
3. 震仓驱离（清除浮动供应）
4. 放量大涨离开（新需求进场，主导市场）

⚠️ 错误做法：
- 盲目靠赌或看指标背离抄底
- 不考虑理论根据，直接通过表象抄底
- 结果：只能是抄在半山腰，造成被套
```

#### Q: 安全抄底四步中的"震仓驱离"是什么？

**问题**：第三节提到的四步抄底中，第3步"震仓驱离"是什么意思？

**解答**：

1. **震仓的定义**
   - 震仓 = CM故意打低价格，目的是清除市场上的浮动供应
   - 浮动供应 = 那些被套的、心理脆弱的散户

2. **具体过程（从第一节图1-3参考）**

   ```
   死角形成之后 → 加速下跌阶段（这就是"震仓"）

   特征：
   - 价格突破趋势线后迅速下跌
   - 下跌速度很快
   - 中间几乎没有反弹
   - 成交量虽然大但主要是恐慌卖单
   ```

3. **震仓的目的**
   - 清除还在死扛的公众（被套的人）
   - 迫使他们割肉卖出
   - 这样CM能以更便宜的价格进场更多
   - 为之后的上涨做准备

4. **在四步抄底中的位置**

   ```
   1. 天量抛售 ← 供应达到极限
   2. 低量测底 ← 供应减少
   3. 震仓驱离 ← CM打低价格，清除浮动卖单
   4. 量大大阳线 ← 新需求进场，底确认
   ```

**关键点**：

- ✅ 震仓是底形成的必要阶段
- ✅ 目的是清洗浮动供应，纯净底
- ✅ 看起来像是继续下跌，但实际是"需求在吸收"
- ✅ 之后会出现放量反弹

---

#### Q: "低量测底"具体是什么？不是有人"轻推"来测试吗？

**问题**：低量测底是用少量的卖盘"轻推"来测试价格是否会急速下跌，看有没有新的供应？

**解答**：

1. **你的理解的问题**
   - ❌ "轻推"听起来像有人主动去测试
   - ❌ 实际上是一个被动现象，不是有人"轻推"

2. **正确理解**

   ```
   天量抛售之后：
   - 大量卖单已经流出了
   - 供应基本消耗殆尽
   - 但价格还在继续下跌

   这时的现象 = "低量测底"：
   - 价格下跌
   - 成交量非常小（没有新卖单流入）
   - 说明什么？
     = 下跌是"余波"或"需求在吸收"
     = 不是新的大卖单压进来
   ```

3. **为什么叫"测底"**

   ```
   市场在"尝试"看是否有更多供应：
   - 如果没有新供应出现 → 说明底就在这附近
   - 如果还有大量新卖单 → 说明还要继续跌

   低量 = 供应已经基本没了
   测底 = 市场在"摸底"，看供应是否真的枯竭
   ```

4. **用供求关系解释**

   ```
   天量抛售：供应>>需求，价格大幅下跌+成交量很大
   低量测底：供应≈0，但还有少量卖单余波
             结果：价格继续下跌，但成交量很小

   这说明：供应已经枯竭，底接近
   ```

**关键点**：

- ✅ **"低量"不是说有人轻轻推**
- ✅ **而是说已经没有大卖单了，供应基本枯竭**
- ✅ 低量测底时如果还有急速下跌，说明下方没有防守的买单，反而是供应枯竭的表现
- ✅ 这个阶段后通常会出现反弹或震仓

#### Q: 安全逃顶是否也有类似的四个确定？

**问题**：安全抄底需要这四个确定（天量抛售、低量测底、震仓、量大大阳线），那么安全逃顶是否也要类似的四个确定？

**解答**：

1. **你的直觉完全正确！**
   - 顶形成确实有镜像逻辑
   - 只是方向相反

2. **安全逃顶的镜像四步**

   ```
   安全抄底（底）         安全逃顶（顶）
   1. 天量抛售              1. 天量拉升（抢购高潮）
      供应达到极限             需求达到极限

   2. 低量测底              2. 低量回调
      供应减弱                 需求减弱

   3. 震仓驱离              3. 震顶
      清除浮动卖单             清除浮动买单（获利盘）

   4. 量大大阳线            4. 量大大阴线
      新需求进场              新供应进场
   ```

3. **具体含义**

   ```
   安全逃顶的四步：

   1. 天量拉升（抢购高潮）
      = 买单大量涌入，价格急速上涨
      = 需求达到最高
      = 需求已经用尽

   2. 低量回调
      = 虽然回调，但成交量很小
      = 没有新的买单跟进
      = 需求衰落的信号

   3. 震顶
      = 高位震荡，成交量逐步衰退
      = 或者虽然有反弹，但力度很弱
      = CM在逐步派发筹码

   4. 量大大阴线
      = 大量卖单涌入
      = 新供应进场
      = 牛市结束，熊市开始
   ```

4. **后续学习内容**
   - 这个问题的详细讲解会在：
     - **第六节**："牛市中怎么看出供应进场了——抢购高潮"
     - **第三章**："威氏逃顶策略"
   - 这些章节会详细讲解派发过程和逃顶信号

**关键点**：

- ✅ **你的思维已经达到了"从本质出发推导"的高度**
- ✅ 底和顶的逻辑是镜像的，本质都是供求关系的变化
