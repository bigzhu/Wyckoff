# 第三节 供求关系

## 📌 节点概述

本节深入讲解供求关系这个最核心的理论。供求关系不仅是解释市场运动的根本原理，也是区别威科夫理论和传统技术分析的关键差异。

---

## 核心要点

供应和需求是经济活动中最基本的原理。任何市场的价格波动都来自于供求关系的变化。我们常见的市场表象（支撑、阻力、趋势等）都是由供求关系的不平衡产生的。

**关键理解**：

- 需求 = 购买力
- 供应 = 流入市场的卖单数量
- 供应>需求 = 下跌趋势
- 需求>供应 = 上涨趋势
- 供应≈需求 = 震荡区（容易亏钱的危险区）

---

## 一、供求关系主导趋势的行为

### 简单例子

假设你要下单买 1000 股某只股票，价格是 100 元一股。但市场在这个价位只有 200 股。

```
你的需求：1000 股
市场的供应：200 股
→ 供不应求

经纪公司为了成交，分别以：
- 101 元成交 300 股
- 102 元成交 400 股
- 103 元成交 100 股

结果：价格上涨产生于供求关系的不平衡
```

### 市场含义

从供求关系理解市场的本质：

```
如果掌握了供求关系，即使十次交易只有五次正确，甚至三四次正确，
我们依然能盈利：
- 通过识别供求关系把风险降到最低
- 一次盈利的交易可以补平多次亏损
- 其他盈利交易就是纯盈利
```

**本书的目标**：通过逐根蜡烛分析，反映价格背后的供求关系。懂得了市场的供求关系，也就懂得了市场的本质。

---

## 二、证券市场的本质和表象

### 表象 vs 本质

**表象**：

- 走势图、技术指标（MACD、均线等）
- 几何图形、K线形态
- 很多人依赖这些指标来交易
- **问题**：不断调整参数，市场又变了，陷入恶循环

**本质**：

- 供求关系（市场的第二层面信息）
- 趋势背后的动力
- 需求和供应的对比

### 正确的判断方法

例如：判断牛市结束需要什么条件？

```
理论基础：
1. 需求减弱
2. 供应大量进入市场
3. 需求没有足够力量继续主导市场

这些理论必须从价量行为中体现出来
我们要从直观的图上解读出这些理论
```

### 实战优势

如果掌握了如何从价量行为细节看出供求关系，我们就可以：

✅ 在市场趋势起来**之前**识别出即将出现的趋势
✅ 在危险来临**之前**做好出场准备
✅ 在市场有大的动作**之前**制订交易计划
✅ **实时掌握市场的脉搏**

**这是威科夫理论与传统技术分析的根本区别**

## 三、供求关系和上涨趋势

### 上涨趋势的本质

在上涨趋势中，价格上涨背后的原因是：**供不应求**

```
现象：
- 在同一价格，卖单无法满足增加的购买力
- 买方只能去买更高价位的股票
- 结果：价格不断抬高

这个过程中：
供不应求 → 需求>供应 → 价格上涨
```

### 牛市停止的信号

要使上涨停止或反转，**供求关系必须有变化**：

```
理论基础：
- 人们的购买力完全得到满足
- 不需要再抬高价格来满足需求
- 牛市开始停止

图表表现：
- 虽然参与买入者众多（高成交量）
- 但上涨没有取得多大进展
- 说明供应已经开始增加
  （来自CM的抛盘，因为他们的利润目标已达到）

市场背景的变化：
供不应求 → 供需平衡 → 供过于求
```

**关键意义**：这种市场本质的变化，**指标和软件无法提前告诉我们**！

### 为什么上涨中会出现回调？

```
原因：需求逐步减弱，市场必须降价来刺激新需求

牛市中回调的关键观察：
↓
这个回调是否引起供应扩大（CM抛盘现象）？

✅ 有供应扩大：
   - 回调对市场造成压力增大
   - 市场要恢复上涨趋势，必须有足够新需求
   - 新需求必须吸收回调带来的巨大卖压
   - 并以持续增加的数量恢复上涨趋势

❌ 如果回调后没有更大的新需求：
   - 主力抛盘会继续
   - 市场逐步进入派发阶段
   - 最终进入熊市
```

## 四、供求关系和下降趋势

### 下降趋势的本质

在下降趋势中，**供应>需求，或需求很弱**

```
卖方视角：
- 找不到足够的客户
- 只能不断降价来刺激需求

买方视角：
- 如果价格没跌到满意的价格区
- 就不会买入
- 等待更低的价格
```

### 下跌趋势停止的条件

要让下跌趋势停止，**需求必须大量出现**：

```
必要条件：
1. 需求大量出现
2. 吸收掉当前市场的供应
3. 卖方找到了充足的买家
4. 不用再降价也能销售

结果：
- 价格跌势得到缓解
- 供应逐步减少
- 价格进入盘整

之后：
- CM在盘整区内开始建仓吸筹
- 或者准备再次派发
- 吸筹和派发都是下一波趋势的准备过程
```

### 安全抄底的理论根据

```
判断抄底的理论基础：
1. 供应枯竭
2. 需求增加并主导市场

能够反映出这个理论的价量关系：
1. 天量抛售（供应达到极限）
2. 低量测底（供应逐步减弱）
3. 震仓驱离（清除浮动供应）
4. 长阳带量离开（新需求进场，主导市场）

⚠️ 错误做法：
- 盲目靠赌或看指标背离抄底
- 不考虑理论根据，直接通过表象抄底
- 结果：只能是抄在半山腰，造成被套
```

## 五、培养供求关系的识别能力

### 最终目标

在**不使用技术指标**的情况下，通过**价格和成交量的细节变化**能够识别市场当时的供求关系。

### 具体做法

```
1. 从价量细节中看出供求关系
2. 把市场理念或供求关系从图上解读出来
3. 实时掌握市场的脉搏

结果：
- 对市场的任何变化，都能在重大行为发生之前发现端倪
- 制订相应的应对计划
```

### 实战应用

以后的走势分析主要围绕：

✅ 供求关系
✅ 结合实例
✅ 进行细节分析

**核心能力**：能够从价格和成交量解读出供求关系，从而解读出价格背后的参与者意图（CM和公众）

#### 测试 1：供求关系与价格趋势的基本关系

**我的问题**：

1. 供求关系与价格趋势的基本关系是什么？简单说一下三种情况
2. 为什么说供求关系是市场的"本质"，而技术指标是"表象"？两者的区别是什么？
3. 指标为什么无法提前告诉我们市场本质的变化？举个例子说明

**你的回答**：

1. 供不应求价格涨，供大于求价格跌。供应≈需求时波动（容易亏钱）

2. 指标只是反应供求关系导致的价格变化的趋势，核心决定价格的还是供求关系

3. 指标存在滞后性，需要价格变化甚至很长时间后才能反应过来。比如牛市顶部，观察量价会发现量大价格变化小，说明CM已经在抛售导致买单全被吃掉了，然而此刻市面上的指标肯定显示依然是牛市要求继续加仓买入

**我的评价和补充**：

✅ **理解正确的部分**：
- 第1点完整准确，理解了三种供求关系对应的价格行为
- 第2点抓住了本质：指标是结果，供求关系是原因
- 第3点非常深刻，用牛市顶部的具体例子说明了指标滞后性，展现了对本节核心论点的深入理解

⚠️ **需要补充的部分**：
- 第3点中"量大价格变化小"的含义可以明确为：这说明CM已在抛售，需求力量在减弱
- 指标滞后的原因在于：指标是对价格历史数据的计算，而供求关系的变化可能先于价格表现

📌 **关键要点**：
- **指标的本质**：是对已发生的价格变化的反应和计算
- **供求关系的本质**：是驱动价格变化的根本动力
- **威科夫的优势**：通过识别供求关系，可以在市场大动作之前提前发现端倪

**我的评价和补充**：

- 第2点完全正确，抓住了"回调伴随的成交量"这个关键的观察点
