# 第一节 聪明钱的看盘顺序

## 核心思想

聪明钱(CM)通过 **价格、成交量、走势速度** 三个工具观察市场，而不是依赖技术指标。核心方法是 **5 步看图法**。

---

## 关键概念

| 术语                           | 定义                               | 说明                                         |
| :----------------------------- | :--------------------------------- | :------------------------------------------- |
| **停止行为 (Stopping Action)** | 努力(成交量) ≠ 结果(价格幅度)      | 经常出现在底部和顶部，是判断转折点的关键信号 |
| **超卖 (Oversold)**            | 快速下跌，无反弹，大成交量         | 意味着反弹即将来临，但**不是进场信号**       |
| **恐慌抛售 (Selling Climax)**  | 下跌趋势中潮水般抛售，成交量达顶峰 | 标志着下跌极端，之后通常会反弹               |
| **吸筹 (Accumulation)**        | CM在底部建仓的过程                 | 需要时间消化浮动供应，是一个反复测试的过程   |
| **死角 (Death Zone)**          | 上升趋势线和水平阻力线的交点       | 价格空间变小，大动作即将来临                 |
| **震仓 (Shakeout)**            | CM故意打低价格清除浮动供应         | 需通过下跌速度和反弹力度区分                 |

---

## 核心逻辑/框架

### 5 步看图法

| 步骤            | 内容                           | 说明                   |
| :-------------- | :----------------------------- | :--------------------- |
| **1. 背景**     | 牛市/熊市/震荡                 | 决定后续形态的解读方向 |
| **2. 价量形态** | 蜡烛长度、成交量高度、走势速度 | 客观记录市场行为       |
| **3. 性质分析** | 走势强弱、形态原因             | 分析市场行为背后的含义 |
| **4. 结论预测** | 推断供求关系和CM意图           | 结合背景做出方向判断   |
| **5. 行动决策** | 进场/出场/持有                 | 根据分析结果制定操作   |

## 案例讲解

### 案例 1：牛市中的停止行为

![图1-1：牛市中的停止行为](../wyckoff_content/OEBPS/Image00002.jpg)

| 分析维度 | 观察                  | 结论             |
| :------- | :-------------------- | :--------------- |
| **背景** | A→B 牛市，成交量递增  | 上涨趋势         |
| **形态** | 小蜡烛 + 小成交量回调 | 没有供应进入市场 |
| **性质** | 价格回落力度很弱      | 停止行为         |
| **预测** | 价格还会尝试上涨      | 趋势延续         |
| **行动** | ✅ **进场买入**        | 顺势交易         |

---

### 案例 2：牛市中的供应扩大

![图1-2：牛市中的供应扩大](../wyckoff_content/OEBPS/Image00003.jpg)

| 分析维度 | 观察                  | 结论         |
| :------- | :-------------------- | :----------- |
| **背景** | C→D 继续上涨          | 牛市背景     |
| **形态** | 大阴线 + 大成交量回调 | 供应进入市场 |
| **性质** | 回调力度 > 上涨力度   | 异常行为     |
| **预测** | 供应强于需求          | 牛市可能反转 |
| **行动** | ❌ **不能买入**        | 等待供应枯竭 |

---

### 案例 3：从底部到吸筹完成

![图1-3：吸筹全过程](../wyckoff_content/OEBPS/Image00004.jpg)

| 阶段         | 形态               | CM行为       | 市场含义             | 操作       |
| :----------- | :----------------- | :----------- | :------------------- | :--------- |
| **A→B**      | 快速下跌 (超卖)    | 准备接盘     | 需求开始吸收抛盘     | ⏳ 观察     |
| **B**        | 恐慌抛售 (天量)    | 开始吸筹     | 下跌极端，供应释放   | ⏳ 观察     |
| **蜡烛1**    | 遇阻回落           | 吸筹中       | 此前被套筹码解套     | ⏳ 观察     |
| **蜡烛4-5**  | 无量上涨           | 吸筹中期     | 需求衰竭             | ⏳ 观察     |
| **死角**     | 波幅变窄           | 准备清洗     | 平衡即将破裂         | 🔔 警惕     |
| **加速下跌** | 快速下跌 (震仓)    | 清除浮动供应 | 吓出最后的不坚定者   | 🔔 准备     |
| **蜡烛7**    | 小蜡烛+大量 (停止) | **吸筹完成** | 供应枯竭，新需求接盘 | ✅ **进场** |

**关键逻辑**：

- **为何蜡烛7可进场？**
    1. **小蜡烛** = 价格跌不动
    2. **大成交量** = 有人大量接盘
    3. **背景** = 已通过震仓清洗了浮动供应

![图1-4：后续走势](../wyckoff_content/OEBPS/Image00005.jpg)
*验证：吸筹完成后，价格开启新一轮上涨。*

## 实战要点

### 信号判断原则

| 信号              | 含义         | 应对               |
| :---------------- | :----------- | :----------------- |
| **停止行为**      | 趋势可能反转 | 准备反向操作或离场 |
| **供应/需求扩大** | 趋势强度改变 | 重新评估市场背景   |
| **吸筹/派发完成** | 趋势即将启动 | 最佳进场/出场点    |

### 常见误区警示

| 误区                 | 后果                 | 正确做法               |
| :------------------- | :------------------- | :--------------------- |
| ❌ **只看技术指标**   | 指标滞后，容易被套   | ✅ **关注价量/供求**    |
| ❌ **看到超卖就抄底** | 无吸筹过程，可能被埋 | ✅ **等待吸筹完成信号** |
| ❌ **放量大涨就追**   | 可能是派发陷阱       | ✅ **观察背景和位置**   |

### 实盘操作流

1. **扫一眼大周期** → 确定背景(牛/熊/震荡)
2. **观察最近蜡烛** → 记录价量变化
3. **快速分析性质** → 力量对比如何？
4. **推断CM意图** → 是他在买还是卖？
5. **决定行动** → 有把握才出手

---

## 核心总结

| 维度         | 要点                                     |
| :----------- | :--------------------------------------- |
| **三大工具** | 价格(蜡烛长度)、成交量、走势速度         |
| **五步法则** | 背景 → 形态 → 性质 → 结论 → 行动         |
| **核心理念** | 供求关系决定趋势；CM行为通过价量留下痕迹 |
| **关键信号** | 停止行为(努力≠结果)是识别拐点的利器      |
| **底部定律** | 没有吸筹过程的底部不是真底               |
