## 第一节 聪明钱的看盘顺序

## 核心思想

聪明钱（CM）通过 **价格、成交量、走势速度** 三个工具观察市场，而不是依赖技术指标。核心方法是 **5 步看图法**。

---

## 📋 5 步看图顺序

| 步骤 | 内容         | 说明                                      |
| ---- | ------------ | ----------------------------------------- |
| 1    | **背景**     | 牛市/熊市/震荡 - 决定后续形态的解读       |
| 2    | **价量形态** | 蜡烛长度、成交量高度、变化速度 - 客观记录 |
| 3    | **性质分析** | 走势强弱、形态原因、可能后果 - 分析含义   |
| 4    | **结论预测** | 推断供求关系和 CM 意图                    |
| 5    | **行动决策** | 进场/出场/持有                            |

---

#### 流程分解

| 阶段     | 形态          | 性质         | CM 行为      | 判断        |
| -------- | ------------- | ------------ | ------------ | ----------- |
| A→B      | 快速下跌      | 超卖         | 准备接盘     | ⏳ 观察     |
| B        | 恐慌抛售      | 天量         | 开始吸筹     | ⏳ 观察     |
| 蜡烛1    | 大幅反弹+上影 | 遇强阻力     | 吸筹中       | ⏳ 观察     |
| 蜡烛2    | 再次受阻      | 供应持续     | 吸筹中       | ⏳ 观察     |
| 蜡烛4-5  | 无量上涨      | 需求衰竭     | 吸筹中期     | ⏳ 观察     |
| 死角     | 波幅变窄      | 平衡破裂前   | 准备清洗     | 🔔 警惕     |
| 加速下跌 | 快速下跌      | 震仓清洗     | 清除浮动供应 | 🔔 准备     |
| 蜡烛7    | 小蜡烛+大量   | **停止行为** | **吸筹完成** | ✅ **进场** |
| 最后阳线 | 强势上涨      | 需求主导     | 允许上涨     | ✅ **确认** |

#### 关键要点

- **为什么 B 不能抄底**：没有看到吸筹完成的信号，市场可能进入长期底部巩固
- **为什么 1-5 不能抄底**：吸筹过程还在进行，资金可能被困在横盘期
- **为什么蜡烛7可以进场**：
  - **小蜡烛** = 价格波动小
  - **大成交量** = 大量卖单流入
  - **不下跌** = 新需求在强势吸收
  - **含义** = 供应枯竭，底部形成完毕

## 🔑 关键概念

### 停止行为（Stopping Action）

**定义**：努力（成交量）≠ 结果（价格幅度）

**两种表现**：

1. **高成交量 + 小蜡烛** = 卖单多，需求在强势承接 → 可能反转向上
2. **大蜡烛 + 小成交量** = 价格跌，卖单少，下跌动力在衰竭 → 可能反转向上

**实战意义**：经常出现在底部和顶部，是判断转折点的关键信号。

---

### 超卖（Oversold）

快速下跌，无反弹，大成交量。通常意味着反弹即将来临，但 **不是进场信号**。

---

### 恐慌抛售（Selling Climax - SC）

下跌趋势中的潮水般抛售，成交量达到顶峰。标志着下跌的极端，之后通常会反弹。

---

### 吸筹（Accumulation）

CM 在底部建仓的过程，需要时间让浮动供应逐步消化。不是看到底就立刻开始，而是一个**反复测试、逐步吸筹的过程**。

---

### 死角（Death Zone）

上升趋势线和水平阻力线的交点。价格被夹在越来越小的空间，大动作即将来临。

---

### 震仓（Shakeout）

CM 故意打低价格，清除市场上的浮动供应（被套散户的割肉盘）。需要通过**下跌速度和反弹力度**来区分是震仓还是真正的熊市。

---

## 💡 实战要点

### 三个最常用的信号

1. **停止行为** → 趋势可能反转
2. **供应/需求扩大** → 趋势强度判断
3. **吸筹/派发完成** → 进场/出场时机

### 常见错误

| 错误           | 后果                   |
| -------------- | ---------------------- |
| 只看技术指标   | 指标滞后，容易被套     |
| 看到超卖就抄底 | 没有吸筹过程，反复被锤 |
| 反弹就认为转牛 | 可能还在吸筹早期       |
| 放量大涨就买   | 可能是 CM 在派发       |
| 只看价格不看量 | 判断不全面             |

### 实盘应用

1. **扫一眼大周期** → 确定背景
2. **观察最近蜡烛** → 记住价量变化
3. **快速分析** → 这个形态说明什么？
4. **推断意图** → CM 在做什么？
5. **决定行动** → 买/卖/等待

---

## 📌 核心总结

| 要素         | 核心含义                              |
| ------------ | ------------------------------------- |
| **5 步法**   | 背景→形态→性质→结论→行动              |
| **3 大工具** | 价格（蜡烛长度）、成交量、走势速度    |
| **停止行为** | 努力≠结果 = 反转信号                  |
| **底部特征** | 需要完整的吸筹过程，不能跳过          |
| **进场时机** | 吸筹完成的信号：停止行为 + 新需求出现 |
| **最大误区** | 把短期反弹当作趋势反转                |
