# 第五节 如何识别供应和需求扩大

## 📌 节点概述

本节讲解如何从价量图表上识别供应和需求的扩大，这是掌握供求关系的关键实战技能。

---

## 核心概念

### 震荡区中的供应和需求扩大

**需求扩大**的表现：

```
位置：震荡区的右手边
形态：放量大涨（放量 + 相对大的涨幅）
含义：大量买单流入市场，价格收高
```

**供应扩大**的表现：

```
位置：震荡区
形态：放量 + 上影线（放量大跌）
含义：卖单流入市场，形成阻力
```

---

## 一、需求吸收阻力供应的判断标准

### 向上突破 vs 向下反转

| 情况         | 理论基础                             | 信号               |
| ------------ | ------------------------------------ | ------------------ |
| **向上突破** | 阻力位没有供应扩大，需求积极吸收供应 | 放量大涨突破       |
| **向下反转** | 阻力位产生大量供应，需求无力吸收     | 高量上影线反弹无力 |

### 关键蜡烛形态分析

**放量大涨（VDC）**：

- 形态：放量 + 大涨
- 含义：需求控制市场
- 应用：底形成支撑，价格回测为进场点
- 图示：图1-9蜡烛1 / 图1-10蜡烛6

**放量大跌（VSC）**：

- 形态：放量 + 大跌
- 含义：供应主导市场
- 应用：顶形成阻力
- 图示：图1-9蜡烛2 / 图1-10蜡烛2
- 离场判断：
  - 放量大跌出现后观察需求能否再次吸收这个供应
  - 若反弹无力/缩量反弹 → 供应压倒需求 → 应该离场
  - 若出现放量大涨 → 需求还在 → 可继续持有但设置止损

**下冲反弹(Spring)**：

- 形态：价格下测支撑后反弹 + 放量
- 含义：需求出现，吸收了供应
- 应用：进场做多的信号
- 图示：图1-9蜡烛4 / 图1-10蜡烛4

## 二、熊市中需求的扩大：恐慌抛售

### 基本特征

```
熊市中的需求扩大 ≠ 需要高成交量
因为：买盘数量少，高成交量来自卖盘扩大（恐慌抛售）

标志：天量大跌（异常高的成交量配上大幅下跌）
含义：巨大的买盘吸收恐慌卖盘（CM在吸筹）
```

### 恐慌抛售现象

**原因**：

- 被套的公众无法承受心理压力
- 在利空消息恐吓下选择抛售
- 卖单被聪明钱（CM）吸收

**含义**：

- 需求的质量好于供应（CM vs 公众）
- 熊市中需求扩大是牛市出现的前提
- 但不是立即进场信号，需要二次测试确认

### 放量滞跌（努力没有结果）

**现象**：

```
蜡烛A：天量大跌（恐慌抛售）
蜡烛B：放量，但跌幅相对减小

分析：
- 成交量：还在扩大（市场很活跃）
- 结果：跌幅在减小（努力没有产生结果）
- 含义：供应开始枯竭，底可能出现
```

---

## 关键要点

### 位置决定含义

```
同样的"放量大涨"：
- 在底 = 放量大涨（VDC）= 支撑形成
- 在顶 = SOS（抢购高潮）= 派发完成信号
```

### 判断阻力突破的实战方法

```
1. 价格接近阻力（蜡烛1）
   → 观察：是否有供应扩大？

2. 阻力位回调测试（蜡烛4、5）
   → 观察：是否缩量？浮动供应是否清除？

3. 二次测试（蜡烛5）
   → 低成交量 = 供应枯竭 = 向上突破可能性大
   → 高成交量 = 供应还多 = 需谨慎

4. 第三次尝试（蜡烛6、8）
   → 放量大涨 = 需求占上风
   → 缺量突破 = 需求不足，价格会回跌
```

### 熊市反转的必要过程

```
第一步：恐慌抛售（天量大跌）
  ↓
第二步：出现放量反弹（需求扩大现象）
  ↓
第三步：观察倾向从空转看涨
  ↓
第四步：需要二次测试确认
  ↓
第五步：观察吸筹过程和浮动供应清除
  ↓
第六步：确认牛市开始
```

**关键提醒**：

- 熊市中第一次放量反弹不是进场点
- 需要吸筹过程 + 浮动供应测试 + 底确认
- 这就是因果关系原则的体现
