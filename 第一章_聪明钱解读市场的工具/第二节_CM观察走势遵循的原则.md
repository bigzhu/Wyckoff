# 第二节 CM 观察走势遵循的原则

## 📌 节点概述

本节介绍聪明钱在观察市场时遵循的三个基本原则：
1. **供求关系原则**
2. **因果关系原则**
3. **努力和结果的关系原则**

这三个原则是威科夫理论的理论基础。

---

## 🔄 三大原则概述

| 原则 | 核心概念 | 应用场景 |
|------|---------|---------|
| **供求关系** | 不平衡产生趋势 | 判断强弱、确定方向 |
| **因果关系** | 因（准备）→ 果（趋势） | 避免盲目抄底，确认反转 |
| **努力和结果** | 努力 ≠ 结果 = 信号 | 识别停止行为，判断拐点 |

---

## 原则 1️⃣：供求关系原则

### 基本概念

趋势的动力来自于**供求关系的不平衡**

```
需求 > 供应  →  上涨趋势
供应 > 需求  →  下跌趋势
需求 = 供应  →  震荡区（危险）
```

### 核心理解

**供求关系最根本的作用**：
- 在交易中，你已经买入后，接下来市场的供求关系变化将决定：
  - 立刻离场（供求关系否定你的判断）
  - 或持有（供求关系支持你的判断）

这意味着你需要**持续评估和衡量市场供求关系的变化**。

---

## 原则 2️⃣：因果关系原则

### 目的

**判断趋势反转**，从而使我们在趋势最初期介入市场，达到降低风险的目的

### 定义

```
因  →  趋势出现前的准备过程（吸筹或派发的震荡区）
果  →  趋势的形成和发展
```

### 核心强调

> **强调因果关系的目的就是提醒大家不要着急抄底。**

### 使用意义

**熟练使用因果关系，可以避免盲目抄底，并使我们在趋势起飞之前的最低风险位置介入市场。**

### 重要提醒

```
任何主要趋势的反转之前，都需要一个过程

这个过程包括：
- 吸筹过程（为牛市做准备）
- 派发过程（为熊市做准备）
- 震荡区交易（整理期）

没有这个过程，接下来形成的趋势不会持续太久

例：公众喜欢抄底，他们在熊市中的第一个反弹就买入
结果：被套或被震仓出局

他们失败的原因：没有等到吸筹过程结束，这个过程就是因，后面产生的趋势就是果
```

---

## 原则 3️⃣：努力和结果的关系原则

### 定义

```
努力  = 成交量（市场流入订单总量）
结果  = 价格的进展幅度

努力没有产生相应结果 = 停止行为 = 危险信号
```

### 主要作用

**告诉我们走势出现了停止行为（Stopping Action）**

### 实际应用

特别是在阻力和支撑价位的时候，我们利用"努力和结果"的理论可以**提前知道后市的方向**。

### 具体案例

#### 在支撑位置的应用

```
场景：价格接触支撑价位

观察点：此时的成交量和价格范围

现象：
- 成交量突然扩大（大量卖单流入）
- 但价格范围很窄（小蜡烛，只跌了一点点）

分析：
- 努力：成交量扩大 = 市场很努力在卖出
- 结果：价格范围很窄 = 没有产生预期的大幅下跌
- 这说明什么？

含义：
这么大量的卖单流入市场，却没有使价格大幅下跌
说明市场的买单流入量远远大于卖单流入量
需求完全截住了下跌势头

这就是一种停止行为！

交易决策：
这种停止行为出现在支撑位，说明支撑有效
可以考虑在这里买入
```

## 📚 三个原则的综合应用

### 判断基础

所有交易判断都要以**供求关系理论为基础**，然后从当时的**价量关系中识别出用于判断的理论根据**，最后采取行动。

### 实例：如何安全地抄底？

#### 理论根据

抄底的理论基础：
1. 供应枯竭（卖单流入量已经很少）
2. 需求增加并主导市场（买单大量流入）

#### 价量表现

能够反映出这个理论基础的价量关系是：

```
步骤1：天量抛售
- 高成交量
- 大幅下跌
- 表明供应在达到极限

步骤2：低量测底
- 成交量萎缩
- 价格继续下跌但力度变弱
- 表明供应在减弱

步骤3：震仓驱离浮动供应
- 快速下跌
- 清除残留的卖单
- 表明需求在承接

步骤4：长阳带量离开底部
- 成交量扩大
- 强势上涨
- 表明需求完全控制市场
```

**只有这些价量行为都出现了，我们才知道抄底的理论根据已经成立。**

#### 错误做法

❌ 如果盲目地靠**赌**或看**指标背离**抄底
❌ 不考虑理论根据而去直接通过**表象**抄底
❌ 结果：只能是抄在**半山腰**，造成被套的局面

## 🎯 供求关系的衡量能力

### 最高目标

培养一种能力：
> 在**不使用技术指标**的情况下，通过**价格和交易量的细节变化**能够识别市场当时的供求关系

### 具体体现

- 通过对图上价量的分析把**市场理念**或**供求关系**解读出来
- 这样我们能**实时掌握市场的脉搏**
- 对市场发生的任何变化，我们能够在**重大行为发生之前发现端倪**
- 并制订相应计划

### 实战优势

以后我们的走势分析主要围绕着**供求关系结合实例进行细节分析**

这意味着我们：
- 能在市场有大的动作之前制订交易计划
- 不用等待指标信号
- 可以抢在公众之前行动
