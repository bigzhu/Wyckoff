# Wyckoff 威科夫交易理论学习笔记

欢迎来到威科夫交易理论的学习之旅。本项目是对《Wyckoff 威科夫方法》这本书的深度学习和总结。

---

## 📚 项目概述

**项目名**：Wyckoff 交易理论学习与研究
**学习对象**：《Wyckoff 威科夫方法》
**学习方式**：逐章逐节深入学习，配合案例讲解和实战应用

### 威科夫理论简介

威科夫理论（Wyckoff Method）是由查尔斯·威科夫（Charles Wyckoff）开创的技术分析体系。它不同于传统的技术指标分析，而是通过分析**价格、成交量和走势速度**，来理解市场背后的**供求关系**和**主力机构的意图**。

**核心思想**：

- 市场由主力机构（聪明钱）和公众组成
- 主力在底部吸筹，顶部派发
- 通过供求关系分析可以识别主力意图
- 在关键位置提前介入，获取最大利润
