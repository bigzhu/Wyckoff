# 第九节 本章核心知识总结

本章深入拆解了[CM(狗庄)](../术语速查手册.md#cm-composite-man)在底部的[吸筹](../术语速查手册.md#accumulation)过程。识别这一过程的核心在于观察**停止行为**与**测试过程**。

## 一、[吸筹](../术语速查手册.md#accumulation)核心信号矩阵

| 信号                                                    | 全称                               | 市场含义 | 关键形态特征                                                                             |
| :------------------------------------------------------ | :--------------------------------- | :------- | :--------------------------------------------------------------------------------------- |
| **[PS](../术语速查手册.md#ps-preliminary-support)**     | Preliminary Support (初次支撑)     | 跌势受阻 | 放量下跌后出现短暂支撑，但往往会被跌破。                                                 |
| **[SC](../术语速查手册.md#sc-selling-climax)**          | Selling Climax (恐慌抛售)          | 极度恐慌 | 极宽的高低价差 + 天量。[CM(狗庄)](../术语速查手册.md#cm-composite-man)利用恐慌通吃筹码。 |
| **[ST](../术语速查手册.md#st-secondary-test)**          | Secondary Test (二次测试)          | 确认底部 | 低量回测 SC 的低点。若量缩价稳，[吸筹](../术语速查手册.md#accumulation)开始的信号。      |
| **[Spring](../术语速查手册.md#spring)**                 | Spring (弹簧效应)                  | 最后一跌 | 跌破支撑后迅速收回。清洗最后的浮动供应。                                                 |
| **[LPS](../术语速查手册.md#lps-last-point-of-support)** | Last Point of Support (最后支撑点) | 强势特征 | 回调的低点不断抬高，回调缩量。                                                           |
| **[SOS](../术语速查手册.md#sos-sign-of-strength)**      | Sign of Strength (强势信号)        | 需求主导 | 大阳线伴随高成交量，突破震荡区或关键阻力。                                               |

---

## 二、[吸筹](../术语速查手册.md#accumulation)阶段判定表

| 阶段       | 目的         | 典型行为                                                                              | 交易策略                                 |
| :--------- | :----------- | :------------------------------------------------------------------------------------ | :--------------------------------------- |
| **阶段 A** | **停止下跌** | 出现 PS -> SC -> [AR(自动反弹)](../术语速查手册.md#ar-automatic-rally-reaction) -> ST | **观望**。等待下跌趋势明确终止。         |
| **阶段 B** | **建立仓位** | 漫长的震荡区，[CM(狗庄)](../术语速查手册.md#cm-composite-man)悄悄吸货                 | **高抛低吸**。在区间边界操作，但需谨慎。 |
| **阶段 C** | **最后测试** | Spring 或 震仓 (Shakeout)                                                             | **最佳进场点 1**。在测试确认后进场。     |
| **阶段 D** | **准备起飞** | SOS (强势上涨) + LPS (缩量回调)                                                       | **最佳进场点 2**。在 LPS 处加仓或进场。  |
| **阶段 E** | **牛市开始** | 离开震荡区，进入上升趋势                                                              | **持仓/加仓**。顺势而为。                |

---

## 三、牛市起步的三大确认

1.  **低点抬高**：随后的回调低点不再触及前低（LPS 出现）。
2.  **量价配合**：上涨放量（需求大），下跌缩量（供应小）。
3.  **突破确认**：价格强力突破震荡区阻力（[JOC](../术语速查手册.md#joc-jump-across-the-creek)），并未再回到区间深处。
