# 第六节 牛市中怎么看出供应进场了：抢购高潮

### 核心思想

抢购高潮（BC）是牛市顶部的关键信号，表面上是需求强势，实质上是CM在派发。理解这一过程，才能在牛市最高点识别出危险信号。

---

## 📋 抢购高潮速查表

| 特征                   | 表现               | 供求含义               | 交易信号   |
| :--------------------- | :----------------- | :--------------------- | :--------- |
| **新高 + 长阳 + 天量** | 放量大涨，创造新高 | CM派发，公众抢购       | ❌ 危险信号 |
| **自然回落**           | 价格出现第一次下跌 | 派发初期，市场自然回调 | ⏳ 警惕     |
| **二次测试反弹无力**   | 小蜡烛 + 缺量反弹  | 供应战胜需求，派发完成 | ❌ 牛市终止 |

---

## 一、抢购高潮的定义与特征

### 什么是抢购高潮

上升趋势结束时，公众因害怕错过最后的机会而疯狂买入。这种疯狂在最高点形成一个"高潮"，之后市场自然回落。

### 典型特征

- **成交量**：陡然剧增（天量出现）
- **涨速**：上涨速度骤然加快
- **蜡烛**：长度（幅度）显著增加
- **价格**：创造新高

### 关键分析

- 蜡烛1：新高长阳 + 天量 = 抢购高潮
- 供应来源：CM开始出货（看到利润目标达到）
- 需求来源：公众疯狂抢购（害怕错过最后一班车）
- 供应质量好于需求质量（机构 vs 散户）

## 二、CM的派发策略

### 为什么长阳和放量在牛市中危险

在健康的牛市中，不希望看到：

- 超长的阳线
- 极端的成交量

这些不是真正的牛市行情，而是**CM的出货手段**。

### 公众的心理陷阱

| 阶段     | 心理状态   | 行为       | 结果       |
| :------- | :--------- | :--------- | :--------- |
| 看到长阳 | 兴奋、渴望 | 急于追高   | 被套高位   |
| 害怕错过 | 恐惧心理   | 疯狂买入   | 成为接盘方 |
| 已至顶部 | 贪婪到极限 | 不在乎陷阱 | 最终被派发 |

## 三、派发完成的确认

### 抢购高潮的完整过程

| 步骤 | 现象                    | 供求含义           | 含义         |
| ---- | ----------------------- | ------------------ | ------------ |
| 1    | 新高长阳天量            | 供应扩大，公众抢购 | 派发**开始** |
| 2    | 自然回落                | 初期派发推进       | 市场调整     |
| 3    | 二次测试反弹            | 公众继续参与       | 测试买力     |
| 4    | 反弹无力（小蜡烛+缺量） | 买力已尽，供应主导 | 派发**完成** |

**关键确认标准**：小蜡烛 + 缺量反弹 = 供应战胜需求 = 熊市开始

## 💡 实战要点

### 识别抢购高潮的唯一标准

**三个条件缺一不可**：

1. ✅ 新高（价格创造新高）
2. ✅ 长阳（大幅上涨）
3. ✅ 天量（极端成交量）

三个条件同时出现 = 抢购高潮 = 派发开始

### 常见错误

| 错误           | 原因           | 后果         |
| -------------- | -------------- | ------------ |
| 长阳放量就追   | 不理解派发本质 | 在最高点买入 |
| 反弹就抄底     | 不等待派发确认 | 被反复锤打   |
| 第一次回调就出 | 过度谨慎       | 错过反弹     |
| 不看成交量     | 只看价格       | 判断不全面   |

### 派发和吸筹的对比

| 特征         | 吸筹（底部）                   | 派发（顶部）                   |
| ------------ | ------------------------------ | ------------------------------ |
| **形态**     | 天量大跌 + 低量测底 + 放量反弹 | 新高长阳 + 自然回落 + 反弹无力 |
| **参与者**   | CM吸收公众卖单                 | CM抛出给公众买单               |
| **关键信号** | 需求出现、供应枯竭             | 供应扩大、需求衰竭             |
| **后市**     | 牛市开始                       | 熊市开始                       |

## 📌 核心总结

### 三个关键认识

**1. 表面强势 = 实质派发**

- 新高、长阳、天量看似强势
- 实际上是CM理性派发、公众盲目接盘
- 这是**力量不对等的交易**

**2. 供应质量 > 需求质量**

- 供应来自机构（CM），有计划有力度
- 需求来自散户（公众），情绪化无续航力
- 最终CM派发力量战胜公众购买力

**3. 派发是过程不是瞬间**

- 抢购高潮是派发**开始**的信号
- 需要看完整过程确认派发完成
- 小蜡烛+缺量反弹才是真正的确认
