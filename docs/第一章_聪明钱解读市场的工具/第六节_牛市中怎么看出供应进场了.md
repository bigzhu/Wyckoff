# 第六节 牛市中怎么看出供应进场了：BC(抢购高潮)

[BC(抢购高潮)](../术语速查手册.md#bc-buying-climax)是牛市顶部的关键信号，表面上是需求强势，实质上是[CM(狗庄)](../术语速查手册.md#cm-composite-man)在派发。理解这一过程，才能在牛市最高点识别出危险信号。

## 一、BC(抢购高潮)的定义与特征

### 什么是 BC(抢购高潮)上升趋势结束时，公众因害怕错过最后的机会而疯狂买入。这种疯狂在最高点形成一个"高潮"，之后市场自然回落。

### 典型特征

-   **成交量**：陡然剧增（天量出现）。
-   **涨速**：上涨速度骤然加快。
-   **蜡烛**：长度（幅度）显著增加。
-   **价格**：创造新高。

| **力量对比** | 供应质量 > 需求质量                                     | 机构出货力度 > 散户接盘力度                             |

## 二、CM的派发策略

### 为什么长阳和放量在牛市中危险？

在健康的牛市中，不希望看到**超长的阳线**和**极端的成交量**。
这些不是真正的牛市行情，而是[**CM(狗庄)**](../术语速查手册.md#cm-composite-man)**的出货手段**。

### 公众的心理陷阱

| 阶段         | 心理状态   | 行为       | 结果       |
| :----------- | :--------- | :--------- | :--------- |
| **看到长阳** | 兴奋、渴望 | 急于追高   | 被套高位   |
| **害怕错过** | 恐惧心理   | 疯狂买入   | 成为接盘方 |
| **已至顶部** | 贪婪到极限 | 不在乎陷阱 | 最终被派发 |

### 常见错误

| 错误               | 原因           | 后果         |
| :----------------- | :------------- | :----------- |
| **长阳放量就追**   | 不理解派发本质 | 在最高点买入 |
| **反弹就抄底**     | 不等待派发确认 | 被反复锤打   |
| **第一次回调就出** | 过度谨慎       | 错过反弹     |
| **不看成交量**     | 只看价格       | 判断不全面   |

## 📌 核心总结

### 三个关键认识

1.  **表面强势 = 实质派发**
    -   新高、长阳、天量看似强势。
    -   实际上是[CM(狗庄)](../术语速查手册.md#cm-composite-man)理性派发、公众盲目接盘。
    -   这是**力量不对等的交易**。

2.  **供应质量 > 需求质量**
    -   供应来自[CM(狗庄)](../术语速查手册.md#cm-composite-man)，有计划有力度。
    -   需求来自散户（公众），情绪化无续航力。
    -   最终[CM(狗庄)](../术语速查手册.md#cm-composite-man)派发力量战胜公众购买力。

3.  **派发是过程不是瞬间**
    -   [BC(抢购高潮)](../术语速查手册.md#bc-buying-climax)是派发**开始**的信号。
    -   需要看完整过程确认派发完成。
    -   小蜡烛+缺量反弹才是真正的确认。
