RedK EVEREX 旨在探索“成交量价格分析 (VPA)”及威科夫“投入与产出”法则，通过对比相对成交量（投入）和相对价格行为（产出），计算“相对流动率 (RROF)”，以检测趋势和反转。

## 核心法则

### VPA 与 投入/产出
结合价格与成交量判断趋势，核心思想是 **大投入应有大产出**。如果不匹配，就是**异常 (不平衡)**。

| 现象             | 描述           | 含义                            |
| :--------------- | :------------- | :------------------------------ |
| **高成交量**     | 市场观点强烈   | 价格变动的动力源                |
| **价涨 + 量增**  | 投入大，产出大 | 趋势持续 (看涨)                 |
| **价跌 + 量增**  | 投入大，产出大 | 趋势持续 (看跌)                 |
| **大投入小产出** | 量大价不变     | **预警**：趋势受阻 (努力无效果) |
| **价动量增**     | 量价配合       | **验证**：趋势确认              |

## EVEREX 使用说明

### 工作原理 (堆叠带)
EVEREX 将成交量与价格强度对比，以 **堆叠带** 形式展示：
*   **下方带**：相对成交量 (投入)
*   **上方带**：相对价格强度 (产出)。*注意：此处“产出”并非单纯的价格涨跌，而是综合了单根 K 线与双根 K 线的收盘价位置、价差范围比等多重因素的复合强度指标。*

状态判断：
*   **平衡**：上下带宽度相当。
*   **不平衡**：一宽一窄 (如量大价不变)。
*   **轻松移动**：量窄价宽。

**RROF (相对流动率 - Relative Rate of Flow)**
类似成交量加权的 RSI。下方显示两条曲线：
*   **灰色线 (RROF Smooth)**：实际的 RROF 数值线 (快速)。
*   **彩色线 (Signal Line)**：RROF 的移动平均线 (慢速)。
*   *注：彩色线的颜色由灰色线是否大于 0 决定。灰色线 > 0 时显蓝色，< 0 时显橙色。*

**实战用法**：
1.  **进场预备 (早信)**：灰色线穿过彩色线 (金叉/死叉) —— 代表动能增强，准备变盘。
2.  **趋势确认 (主信)**：灰色线穿越 0 轴 (彩色线同时变色) —— 代表多空主导权正式易手，趋势确立。
3.  **持仓/离场**：
    *   **持仓**：灰色线保持在彩色线外侧 (开口发散)。
    *   **减仓**：灰色线回调击穿彩色线 (动能衰减)。

### 关键标记 (Markers)
图表自动标记两种特殊状态 (基于价格强度/成交量强度的比率)：

| 标记名称              | 图标        | 比率阈值 | 现象            | 含义 (威科夫逻辑)                                                  |
| :-------------------- | :---------- | :------- | :-------------- | :----------------------------------------------------------------- |
| **轻松移动** (EoM)    | ▲/▼ 三角形  | > 120%   | **量小 价大变** | **不努力就有效果** (Result without immense Effort)<br>阻力最小方向 |
| **压缩/蹲踞** (Squat) | ⚪️/➕ 圆/十字 | < 50%    | **量大 价不变** | **努力无效果** (Effort without Result)<br>遭遇强阻力/支撑          |

*(注：比率在 80%-120% 视为合理平衡；50%-80% 视为漂移/无特殊意义)*

### 常见问题与进阶调整
*   **信号过多 (False Signals)**：
    *   **方法一 (常规)**：在设置中增大 **Lookback Length** (例如 20 -> 50)，平滑基准线。
    *   **方法二 (进阶 - 修改阈值)**：若方法一无效，说明默认阈值太宽松。需点击指标源代码 `{}` 修改：
        *   找到 `is_Compression = ...` 和 `is_EoM = ...` (约 130 行)。
        *   **减少 Squat**：将 `50` **调小** (如 `30`)，仅标记极度滞涨。
        *   **减少 EoM**：将 `120` **调大** (如 `150`)，仅标记极度轻盈。
