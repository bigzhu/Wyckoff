# 第二节 Spring 的位置和种类

## 一、[Spring](../术语速查手册.md#spring) 的典型位置

识别 [Spring](../术语速查手册.md#spring) 的前提是判断其所处的位置。只有在正确的背景下，[Spring](../术语速查手册.md#spring) 才是高质量的交易信号。

| 位置类型                                                             | 市场含义                                                               | 动作建议                                                                     |
| :------------------------------------------------------------------- | :--------------------------------------------------------------------- | :--------------------------------------------------------------------------- |
| **牛城市震荡区底部**                                                 | 价格回调，[Spring](../术语速查手册.md#spring) 出现预示调整结束。       | **进场**（如果是小量低位弹簧）。                                             |
| **[JOC](../术语速查手册.md#joc-jump-across-the-creek) 突破后的回测** | 价格重回突破点，验证支撑有效性。                                       | **进场**。                                                                   |
| **[吸筹](../术语速查手册.md#accumulation)阶段尾声**                  | [CM](../术语速查手册.md#cm-composite-man) 制造恐惧，收购最后低价筹码。 | **进场**（确认 [SOS](../术语速查手册.md#sos-sign-of-strength) 后回测进场）。 |
| **需求柱（Demand Bar）底部**                                         | 垂直需求柱的底部遭到测试。                                             | **进场**。                                                                   |
| **熊市/下跌趋势中**                                                  | 诱导抄底者的陷阱。                                                     | **禁止操作**。                                                               |
| **连续失败的 [Spring](../术语速查手册.md#spring)**                   | 需求持续无法维持，市场本质发生变化。                                   | **预警信号**（平仓/离场）。                                                  |

## 二、[Spring](../术语速查手册.md#spring) 的三种基本类型 (图 4-2)

根据突破支撑时的**蜡烛力度**与**成交量**，可分为三种类型：

![图 4-2：[Spring](../术语速查手册.md#spring) 的三种类型](../wyckoff_content/OEBPS/Image00081.jpg)

### 1. 供应沉重型 (左)
- **特征**：突破支撑时，蜡烛长、成交量大。
- **解析**：支撑位存在大量卖单，需求无法吸收。
- **后果**：价格通常继续下跌寻找新的需求。在[吸筹](../术语速查手册.md#accumulation)右手边称为“[震仓](../术语速查手册.md#shakeout)”。
- **策略**：**观望**，等待**[二次测试](../术语速查手册.md#st-secondary-test)**确认。

### 2. 完美缩量型 (中) —— **最佳进场点**
- **特征**：突破支撑时，蜡烛极小、成交量极小。
- **解析**：向下突破时缺乏供应，需求只需少量介入即可推高价格。
- **策略**：**直接进场**。

### 3. 量增需测型 (右)
- **特征**：突破支撑时，蜡烛和成交量略微扩大。
- **解析**：存在一定供应，需求需要更多力量才能反转。
- **策略**：**等待二次测试**。若二测时蜡烛/量极小，说明供应耗尽，可进场。

## 三、Spring 的衍生与失败

### 1. 强力下影线 (成功的衍生型)
- **特征**：高成交量 + 小实体 + 收盘于 50% 之上的长下影线。
- **背后含义**：高量代表卖方的巨大努力，但价格收高说明这些卖盘被强劲的需求**全部化解并吸收**。性质等同于强力阳线。

### 2. [Spring](../术语速查手册.md#spring) 失败
- **特征**：形成 Type 2 或 Type 3 后，价格未出现持续上涨，成交量随上涨递减。
- **解析**：弹簧“无力”，需求无法维持涨势。
- **后果**：价格通常会跌破支撑继续下行。常发生于熊市或强供应背景。
