# 第五节 Spring 总结

## 核心要义

Spring（弹簧效应）不仅是一个交易信号，更是[**CM(狗庄)**](../术语速查手册.md#cm-composite-man)**（聪明钱）探测市场供应与需求力量对比的战略手段**。

---

## 一、成功的三个支柱

### 1. 位置优先
- **强支撑背景**：Spring必须发生在前期已经建立的强支撑区，或者需求强劲的价位附近。

### 2. 背景决定方向
- **上升趋势**：此时的Spring成功率最高，是顺向加仓或进场的良机。
- **下降趋势**：下降趋势中的Spring往往会失败，这对于专业交易员来说反而是**反手做空**的信号。

### 3. 供求行为验证
- **低量突破**：突破支撑时没有大量卖盘跟随（供应耗尽）。
- **反弹高度**：Spring引发的反弹若无法突破区间顶部，说明需求不足，价格可能再次回测或跌破支撑。

---

## 二、CM(狗庄)的探测逻辑 (实战视角)

| 阶段     | [CM(狗庄)](../术语速查手册.md#cm-composite-man)的动作 | 市场反馈                       | 我们的对应策略                     |
| :------- | :----------------------------------------------------- | :----------------------------- | :--------------------------------- |
| **测试** | 制造跌破支撑的假象                                     | 观察是否有放量卖盘             | 如果是缩量突破，说明供应稀少。     |
| **吸纳** | 确认供应枯竭后迅速买入                                 | 价格迅速反弹，不给公众思考时间 | **跟随进场**，止损设在突破最低点。 |
| **确认** | 推动价格突破区间顶部                                   | 形成持续上涨波段               | 持仓或在回测时加仓。               |


## 三、Spring 成功率判定矩阵

| 维度           | 高胜率Spring(✅ 做多) | 低胜率/失败Spring(❌ 观望或做空) |
| :------------- | :-------------------------------------------------- | :------------------------------------------------------------- |
| **大趋势**     | **上升趋势中** (牛市回调)                           | **下降趋势中** (熊市下跌)                                      |
| **支撑强度**   | 历史强支撑，多次验证有效                            | 临时支撑，未经测试                                             |
| **突破成交量** | **极低** (供应枯竭)                                 | **极高** (供应涌出)                                            |
| **收盘位置**   | 迅速收回支撑之上，收于高位                          | 收于支撑之下或中低位                                           |
| **跟随行为**   | 次日出现阳线跟随                                    | 次日继续下跌或横盘弱势                                         |

## 📈 Spring 交易法则逻辑总结图

```mermaid
graph TD
    Start[价格跌破水平支撑位] --> SupplyCheck{成交量是否显著扩大?}
    SupplyCheck -- 是 --> FollowThrough{随后是否有价格跟随下跌?}
    FollowThrough -- 是 --> Failed["确认为支撑破位: 做多失败/离场"]
    FollowThrough -- 否 --> Absorption["努力没有结果: 供应被吸收"]
    SupplyCheck -- 否 --> SuccessProbability["供应趋于耗尽: 高胜率 Spring"]

    SuccessProbability --> ReboundCheck{反弹力度如何?}
    Absorption --> ReboundCheck

    ReboundCheck -- "强: 创新高/突破阻力" --> Bullish"确认强势: [吸筹"结束/趋势恢复]
    ReboundCheck -- "弱: 无法突破前高" --> Warning["警告: 需求缺失/可能再次测底"]
```

---

## 📌 结语

- **不要迷信图形**：图形只是表象，背后的**市场语言**（供求逻辑）才是真理。
- **让市场指导你**：[CM(狗庄)](../术语速查手册.md#cm-composite-man)非常清楚公众的恐惧心理（看到破位不敢动），我们要站在[CM(狗庄)](../术语速查手册.md#cm-composite-man)的视角，利用低风险的支撑位进行博弈。
- **危机管理**：进场后 3-5 根蜡烛内若无预期表现，应无条件离场。
