# 第七节 Spring 失效与进场判定

## 核心思想

本节通过实战案例深入辨析 **Spring** 在不同背景下的命运。核心逻辑在于：**不要被图形迷惑**。在卖盘控制的背景下，**Spring** 往往是诱多陷阱；只有在回调深度不超过 50% 且需求主导的背景下，**Spring** 才是高胜率的进场信号。

---

## 一、案例 6：需求背景下的 50% 回调 (图 5-7, 5-8)

### 1. 案例分析 (回调深度与吸收)

![图 5-8：50% 回调与停止行为分析](../wyckoff_content/OEBPS/Image00108.jpg)

- **停止行为 (D)**：量增价缩，说明上涨遇阻。
- **无量回调 (E)**：从 D 开始的回调成交量迅速缩小，说明没有供应参与。
- **牛市守则**：只要价格挺在 50% 位置之上（如 AB 波段所示），牛市逻辑即未受威胁。

#### 50% 原则与趋势强度

```mermaid
graph TD
    A["价格上涨波段"] --> B["出现回调"]
    B --> C{"回调深度?"}
    C -->|"超过 50%"| D["弱势: 趋势可能转折"]
    C -->|"小于 50%"| E["强势: 牛市逻辑仍在"]
    E --> F["观察 [停止行为]"] --> G["准备做多"]
```

### 2. 实战结果验证 (图 6-9)

![图 6-9：吸收卖盘后上扬](../wyckoff_content/OEBPS/Image00140.jpg)

- **验证**：蜡烛 E 处的威胁被需求立刻吸收。DE 段的本质是短暂的吸收过程，随后的上扬确认了需求控制权。

## 二、案例 20：趋势转折点的无需求反弹 (图 5-26)

### 1. 案例分析 (死角压力与测试)

![图 5-26：死角突破与弱势反弹](../wyckoff_content/OEBPS/Image00126.jpg)

- **背景转变**：放量阴线突破上升趋势线和死角。
- **反弹虚弱 (1)**：连续 4 个小蜡烛反弹，成交量虽在最后增加，但收盘在低点。这是典型的**阻力位需求衰竭**。

### 2. 实战结果验证 (图 6-26)

![图 6-26：支撑破位与下跌](../wyckoff_content/OEBPS/Image00157.jpg)

- **结论**：预期的支撑被放量跌破。支撑位 ([JOC](../术语速查手册.md#joc-jump-across-the-creek) 回测点) 的突破过程成交量扩大，且由于随后反弹无力，确认了熊市背景的延续。

## 三、案例 23：供应控制下的 Spring 失效 (图 5-30)

### 1. 案例分析 (BC 抢购高潮后的陷阱)

![图 5-30：跳空阴线与弱势反弹](../wyckoff_content/OEBPS/Image00130.jpg)

- **背景预判**：背景中出现了跳空阴线且放量，说明卖压增强。
- **Spring 逻辑 (1)**：虽然 1 处是标准的 **Spring** 形状，但由于背景由供应控制，其成功率骤降。

#### 背景决定 Spring 命运

```mermaid
graph TD
    A["背景判断"] --> B{"谁控制市场?"}
    B -->|"卖盘控制 (供应线完整)"| C["出现 [Spring] 信号"]
    C --> D["诱多陷阱概率大"]
    D --> E["需求无法持续跟随"] --> F["[Spring] 失败: 破位大跌"]
```

### 2. 实战结果验证 (图 6-29)

![图 6-29：Spring 失败破位](../wyckoff_content/OEBPS/Image00160.jpg)

- **结论**：**Spring** 后出现增量阴线并跌破趋势线，需求毫无抵抗。这证明在**卖盘背景下，Spring** 经常失败，甚至可能成为做空信号。

## 四、案例 24：细节锁死——需求与供应的生死线 (图 5-31)

### 1. 案例分析 (力量对比与多空决战)

![图 5-31：阳线与阴线的博弈细节](../wyckoff_content/OEBPS/Image00131.jpg)

- **行为改变 (2)**：放量下挫（蜡烛 2）明显确认了供应在持续。
- **细节观测**：2 之后的长阳反弹未创新高，随后 3 处的阴线收盘于底部，说明在高位存在重重压制。

### 2. 实战结果验证 (图 6-30)

![图 6-30：冰线破位导致崩盘](../wyckoff_content/OEBPS/Image00161.jpg)

- **结论**：蜡烛 5 最终跌破**冰线**。[CM](../术语速查手册.md#cm-composite-man) 撤掉支柱后，价格在无需求反弹 (6) 后陷入崩盘。**冰线**的贯穿意味着原趋势的彻底终结。
