# 前言：用市场的自身行为判断走势

## 1. 核心理念：表象 vs 本质

- **表象 (Trap)**: 技术指标、几何图形、波浪理论等。
- **本质 (Truth)**: 价格、成交量、[CM](./术语速查手册.md#cm-composite-man)的意图。

::: tip 提示
价格波动是由市场参与者的行为造成的，因为[**CM**](./术语速查手册.md#cm-composite-man)(Composite Man)在幕后操纵。
:::

## 2. 市场的真相

市场的残酷来自[**CM**](./术语速查手册.md#cm-composite-man)的操纵。

- **操纵手段**: 利用资金优势制造假象（如诱多、诱空）。
- **目的**: 引诱公众在错误的时机进场，通过“剪羊毛”获利。
- **应对**: 放弃预测，专注于识别图表中[CM](./术语速查手册.md#cm-composite-man)留下的供求关系痕迹。

## 3. CM(狗庄)是谁？

| 角色                                         | 描述                                     | 别名               |
| :------------------------------------------- | :--------------------------------------- | :----------------- |
| **[CM](./术语速查手册.md#cm-composite-man)** | 掌握庞大资金，能短期控制走势的机构或大户 | 主力、庄家、聪明钱 |
| **公众**                                     | 资金分散，容易受情绪和消息影响的散户     | 韭菜、羊毛         |

**[CM](./术语速查手册.md#cm-composite-man)的行为模式**:
- **吸筹**: 在底部恐慌时买入，此时图表枯燥无味。
- **派发**: 在顶部贪婪时卖出，配合利好消息。
- **震仓**: 制造剧烈波动清洗浮筹。

## 4. 聪明钱 vs 公众

![聪明钱和公众的区别](./wyckoff_content/OEBPS/Image00001.jpg)

| 对比项       | 公众 (被收割者)      | 聪明钱 (专业人士)       |
| :----------- | :------------------- | :---------------------- |
| **决策依据** | 消息、传言、技术指标 | **价格、成交量、速度**  |
| **进场时机** | 看到指标信号时       | 供求关系确认时          |
| **风险控制** | 无 (容易套牢)        | **严格的危机管理**      |
| **关注点**   | 表象信息             | **市场自身行为 (供求)** |

## 5. 本书的学习目标

通过本书，你将学会如何像[CM](./术语速查手册.md#cm-composite-man)一样思考：

1.  **抛弃**: 小道消息、媒体观点、滞后的技术指标。
2.  **掌握**: 供求关系分析、[CM](./术语速查手册.md#cm-composite-man)的操作意图识别。
3.  **实践**: 在趋势开始前行动，在零和博弈中站在赢家的一边。
